## 一、矛盾论与辩证法

### 判断主要矛盾的方法

1. **因果链条法**：画出问题的因果链条，处于链条上游、影响最多下游节点的那个矛盾，往往是主要矛盾
2. **杠杆效应法**：解决哪个矛盾能带来最大的连锁反应？
3. **紧迫度×重要度**：综合考虑当前最尖锐和最根本的矛盾
4. **反证法**：如果这个矛盾消失了，问题还存在吗？不存在了就是主要矛盾

### 矛盾的主要方面与次要方面

- 事物的性质，主要地是由取得支配地位的矛盾的主要方面所规定的
- 关注量变到质变：次要方面在不断积累力量

**转化条件分析模板**：
```
当前状态：矛盾 A 中，X 是主要方面，Y 是次要方面
X 的力量来源：______
Y 的力量来源：______
转化条件：当 ______ 发生时，Y 可能取代 X 成为主要方面
转化信号：______ （可观测的早期指标）
应对预案：______
```

### 矛盾的同一性与斗争性

- **同一性**：矛盾双方互为存在条件，在一定条件下互相转化
- **斗争性**：矛盾双方相互排斥、相互对立，推动事物发展
- 增强（邓小平）：并非所有矛盾都要通过斗争解决，有些矛盾可以通过**制度设计实现兼容**

### 量变与质变

- 量的积累到一定程度引起质的飞跃
- 判断趋势：当量变积累到一定程度，质变必然到来
- 个人成长：不要期望线性进步，接受曲折是正常的发展路径

## 二、实践论与调查研究

### 认识的三个阶段

1. **感性认识**：通过直接接触事物获得的表面的、片面的认识
2. **理性认识**：将感性材料加以"去粗取精、去伪存真、由此及彼、由表及里"的改造
3. **回到实践**：用理性认识指导新的实践，在实践中检验、修正、发展

### 调查研究的具体方法

**怎样开调查会**：
> "参加调查会最好有三五人。"——毛泽东

- 调查典型分三种：先进的、中间的、落后的。每类调查两三个即可知一般情形
- 都必须自己亲身去做

**怎样让对方说真话**：
> "主要的一点是要和群众做朋友，而不是去做侦探。群众不讲真话，不怪群众，只怪自己。"——毛泽东

**详细占有材料，抓住要点**：
> "材料是要搜集得愈多愈好，但一定要抓住要点或特点（矛盾的主导方面）。"——毛泽东

### 分析与综合的方法

> "第一步观察大体轮廓。第二步，用分析方法把各部分细细研究。第三步再用综合法得出整体的认识。"——毛泽东

这就是"四面受敌"法：把研究对象分成政治的、经济的、文化的、军事的四个部分来研究。

## 三、邓小平思维模型

### 1. 实事求是检验法（"猫论"）

不看标签看效果，用实际结果而非意识形态标签来评判一切。

**核心**："黄猫、黑猫，只要捉住老鼠就是好猫。"——判断标准在于实际效果，不在于名义上的"主义"。

**与毛泽东的区别**：毛泽东的实事求是更偏重"调查研究"的方法论层面，邓小平的实事求是更偏重"结果导向"的决策层面。

### 2. 摸石头过河法（渐进试错法）

没有现成经验可循时，先小范围试验，成功了再推广，失败了就纠正。

**核心**：
- "看准了的，就大胆地试，大胆地闯。"
- "允许看，比强制好得多。"
- "不搞争论，是我的一个发明。不争论，是为了争取时间干。"

**应用**：经济特区从深圳一个小渔村试验，成功后扩展到14个沿海城市。

### 3. 发展硬道理法（"台阶论"）

发展是解决一切问题的根本，但发展不是匀速的，而是波浪式前进、隔几年上一个台阶。

**核心**：
- "发展才是硬道理。"
- "低速度就等于停步，甚至等于后退。"
- "抓住时机，加速搞几年，发现问题及时加以治理，尔后继续前进。"

**与毛泽东的区别**：毛泽东的发展观强调"多快好省"，容易走向冒进；邓小平强调"讲求效益"，发展速度要与质量、稳定协调。

### 4. 两手平衡法

物质文明和精神文明、改革开放和打击犯罪——必须同时抓，两手都要硬。

**核心**："不讲多劳多得，不重视物质利益，对少数先进分子可以，对广大群众不行，一段时间可以，长期不行。如果只讲牺牲精神，不讲物质利益，那就是唯心论。"

### 5. 稳定压倒法

没有稳定的环境，什么都搞成。

**核心**：
- "中国的问题，压倒一切的是需要稳定。"
- "动乱一下，就耽误好多年，不是三年五年能恢复起来的。"
- 稳定不是僵化：稳定是为改革创造条件

### 6. 一国两制创造法（共存兼容法）

面对看似不可调和的矛盾，不追求一方消灭另一方，而是设计一种框架让对立双方各得其所。

**这是对毛泽东矛盾论的重大发展**：承认某些矛盾不必通过斗争解决，可以通过制度设计实现兼容。

### 7. 邓小平如何修正毛泽东思想

| 维度 | 毛泽东晚年 | 邓小平修正 |
|------|-----------|-----------|
| 主要矛盾 | 以阶级斗争为纲 | 以经济建设为中心 |
| 动力观 | 精神万能、政治挂帅 | 承认物质利益的基础性 |
| 所有制 | 一大二公 | 多种经济形式并存 |
| 对外关系 | 自力更生为主 | 对外开放 |
| 治理方式 | 群众运动 | 制度建设、法制 |
| 矛盾处理 | 斗争性为主 | 兼容性（一国两制） |
| 发展节奏 | 一万年太久只争朝夕 | 台阶式发展 |

## 四、习近平思维能力

### 1. 底线思维

凡事从最坏处着眼、向最好处努力。

**操作**：先想清楚最坏结果能否承受，再决定是否行动。把困难估计得更充分一些。

**口诀**："从最坏处着眼，向最好处努力。"

### 2. 系统观念

把事物看作相互联系的有机整体。

**操作**：做任何重大决定前，先画出所有关联方和连锁反应。处理好发展和安全、整体和局部、长远和短期、重点和非重点的关系。

**口诀**："前瞻思考、全局谋划、战略布局、整体推进。"

### 3. 问题导向

从实际问题出发，把发现问题、分析问题、解决问题作为出发点。

**操作**：哪里有问题，哪里就是改革重点。不回避矛盾，不掩盖问题。

**口诀**："哪个领域问题突出，就把哪个领域作为重点。"

### 4. 战略思维

从全局、长远、大势上作出判断和决策。

**操作**：区分"战略层"和"策略层"，战略上坚定，战术上灵活。

**口诀**："战略坚定性与策略灵活性相结合。"

### 5. 精准思维

对症下药、因地制宜，用绣花功夫把工作做细做实。

**操作**：分析问题要精确到具体环节，不要泛泛而谈。

**口诀**："对症下药，因地制宜，绣花功夫。"

### 6. 守正创新

在坚守根本原则的基础上大胆创新。

**操作**：在坚持核心价值观的前提下，大胆尝试新方法。既不走封闭僵化的老路，也不走改旗易帜的邪路。

### 7. 先立后破

改革旧事物前，先确保新方案已经成熟可靠。

**操作**：不要先拆旧房子再找新住处。传统能源退出必须建立在新能源安全可靠的替代基础上。

**口诀**："新方案成熟前不拆旧框架。"

### 8. 核心智慧速查表

| 场景 | 思维方法 | 核心口诀 |
|------|----------|----------|
| 面对不确定性 | 底线思维 | 从最坏处着眼，向最好处努力 |
| 多因素交织 | 系统观念 | 前瞻思考、全局谋划 |
| 找不到突破口 | 问题导向 | 哪里有问题，哪里就是重点 |
| 长远规划 | 战略思维 | 战略坚定 + 策略灵活 |
| 两难选择 | 辩证思维 | 在危机中育新机 |
| 改革转型 | 先立后破 | 新方案成熟前不拆旧框架 |

## 五、马克思恩格斯思维工具

### 1. 历史唯物主义

一切社会变迁的根源在于物质生产力与生产关系的矛盾运动。

**日常应用**：分析任何社会现象，先问"利益结构是什么"而非"官方说法是什么"。判断一种理论/政策的真假：看它对谁有利。

### 2. 异化理论

你创造的东西反过来成为控制你的力量。

**日常应用**：
- 理解"内卷"：越努力竞争，整体处境越差
- 识别"数字异化"：你发的内容变成平台收割你注意力的工具
- 警惕"消费异化"：你以为在享受商品，其实是商品在定义你的身份

### 3. 剩余价值分析

资本利润的唯一来源是工人劳动创造的价值超出其工资的部分。

**日常应用**：看清工资谈判的本质——你创造的价值远大于你拿到的钱。理解加班文化的实质——延长剩余劳动时间。

### 4. 商品拜物教批判

人们崇拜商品/金钱，忘记了背后是人的劳动。

**日常应用**：穿透品牌光环，问"实际使用价值是什么"。理解金融泡沫——只看到数字涨跌，忘记背后实体经济。

### 5. 生产力-生产关系矛盾运动

技术和生产力不断长大，所有制和制度就像鞋。鞋小了就得换。

**日常应用**：理解AI时代的制度冲突。关注生产力发展方向，不要固守即将被淘汰的旧模式。

### 6. 周期思维

经济危机不是偶然，是系统的必然产物。繁荣→生产过剩→危机→停滞→复苏，周而复始。

**日常应用**：不要在繁荣期忘记储备，在危机期忘记机会。"这次不一样"永远是错觉。

## 六、现代应用案例库

### 商业竞争中的"农村包围城市"

创业公司面对行业巨头：不在巨头的主场硬拼，先在边缘市场建立根据地。拼多多从下沉市场起步，快手从农村用户起步。

### 个人职业发展中的"持久战"

25-30岁是战略防御期（积累能力），30-35岁是战略相持期（找到根据地），35岁以后进入战略反攻期。相持阶段最熬人，很多人在这个阶段放弃。

### 用"异化思维"理解内卷

越努力竞争，整体处境越差。程序员996写代码，代码成了控制自己的系统。要跳出内卷，关键是**找到差异化的位置**，而不是在同一维度上拼命卷。

### 用"底线思维"做重大决策

创业前先想清楚：最坏结果是什么？能承受吗？如果最坏结果是"回到打工状态"，那可以试；如果最坏结果是"倾家荡产"，那就要慎之又慎。

### 用"利益结构分析"理解职场

你的工资不是由你"值多少"决定的，而是由劳动力市场供需决定的。当某个技能供给过剩，再优秀也拿不到高薪。看清这一点，就知道该往哪个方向积累。

### 用"先立后破"做职业转型

不要裸辞后再找工作。在职状态下先找到新方向、积累新能力、拿到新offer，再离开旧岗位。新方案成熟前不拆旧框架。

### 谈判中的"统一战线"

第一步：分清谁是盟友、谁是对手、谁是中间派。第二步：在主要矛盾上与盟友结盟。第三步：对中间派争取团结，对对手集中火力。"又团结又斗争"——不是无原则的和稀泥。
