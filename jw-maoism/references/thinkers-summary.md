# 思想家速查表（v7 · 五大思维大法架构）

## 五大思维大法总览

```
矛盾分析法（母模型）
├── 统一战线（分清敌我友）
├── 穿透表象（物的关系遮蔽人的关系）
└── 利益结构分析（谁在获利谁在劳动）

实践认识循环（母模型）
├── 调查研究（没有调查就没有发言权）
├── 实事求是检验（不看标签看效果）
└── 批评与自我批评（照镜子洗洗澡正衣冠）

持久战略（母模型）
├── 阶段分析（防御→相持→反攻）
├── 台阶式发展（波浪式前进）
└── 周期思维（繁荣→危机→复苏）

根据地策略（母模型）
├── 农村包围城市（从薄弱处切入）
├── 集中兵力（伤其十指不如断其一指）
└── 先立后破（新方案成熟前不拆旧框架）

群众路线（母模型）
└── 从群众中来，到群众中去
```
