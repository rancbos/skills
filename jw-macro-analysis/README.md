# jw-macro-analysis

中国宏观经济周期判断系统

## 简介

中国宏观经济周期位置判断工具。基于24本周期理论著作，通过四周期嵌套模型（康波60年+房地产20年+朱格拉10年+基钦3年）和五维度指标体系，判断当前中国社会环境所处的周期阶段。

触发场景：当用户询问经济周期位置、宏观走势判断、周期拐点识别、大类资产配置建议时使用。
