## 一、马克斯核心量化数据总览

### 1.1 股市长期统计规律（1970-2016年，47年数据）

| 指标 | 数值 | 来源章节 |
|------|------|---------|
| 标普500长期年均收益率 | **约10%** | 第12章 |
| 最高年度收益率 | **+37%** | 第12章 |
| 最低年度收益率 | **-37%** | 第12章 |
| 处于"正常"区间（8%-12%）的年份数 | **仅3年**（平均16年才遇到1年） | 第12章 |
| 大幅偏离正常水平（>30%或<-10%）的年份 | **13年**，占比超过1/4 | 第12章 |
| 1975-1999年标普下跌年份数 | **少数年份**，无一年跌幅超7.5% | 第4章 |
| 1975-1999年涨幅超15%的年份 | **16年** | 第4章 |
| 1975-1999年涨幅超30%的年份 | **7年** | 第4章 |

**核心发现：** 股市表现符合"正常水平"绝对不是正常现象，所谓的常态其实是非常态。极端年份往往成群结队连续出现，间隔仅一两年。

### 1.2 经济周期量化参数

| 指标 | 数值 | 说明 |
|------|------|------|
| 美国GDP长期平均增长率 | **2%-3%/年** | 第4章 |
| GDP年增长率正常区间 | **-2%至+5%** | 第4章 |
| GDP触及极端值的频率 | **每十年最多一次** | 第4章 |
| 企业盈利波动 vs GDP波动 | **2-3倍甚至更大** | 第6章（杠杆放大效应） |
| 企业盈利周期放大倍数 | 经营杠杆+财务杠杆共同作用 | 第6章 |

### 1.3 杠杆放大效应的量化示例

**经营杠杆示例（出租车公司）：**
- 客流量增长20% → 销售收入增长20% → 但总成本增幅低于收入增幅 → 营业利润增长幅度 > 20%

**财务杠杆示例：**
- 资本结构：15,000美元权益 + 15,000美元债务（年利息1,500美元）
- 营业利润从3,000美元降至2,000美元（下降33%）
- 净利润从1,500美元降至500美元（下降67%）
- **财务杠杆将波动放大1倍**
