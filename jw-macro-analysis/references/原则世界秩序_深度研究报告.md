# 《原则：应对变化中的世界秩序》深度研究报告

> 基于瑞·达利欧原著的系统性提炼与分析
> 聚焦：大周期框架、帝国兴衰规律、18个决定因素、中国在大周期中的位置

---

## 第一部分：大周期理论框架

### 1.1 核心思想：历史是一台永动机

达利欧的核心洞见是：**历史通过相对明确的生命周期演进，随着一代人向下一代人的过渡而逐步发展**。这些生命周期汇聚在一起，构成一个自古至今、包罗万象的整体历史演进过程。同样的事件受大致相同的原因驱动而反复发生，同时也在不断演进。

> "未来的时代将与我们有生之年所经历的时代完全不同，但与历史上的许多时代有着相似之处。"——达利欧

**三个核心命题：**
1. 没有任何政治体制、经济体系、货币或帝国可以永远存在
2. 从一个极端到另一个极端的变化是常态，而不是例外
3. 在一个世纪里没有经历至少一个和平与繁荣时期以及一个萧条、革命与战争时期的情况极为罕见

### 1.2 进化与周期的螺旋上升

达利欧将历史发展概括为**进化与周期的双重结构**：

- **进化**是宇宙中最大的也是唯一永恒的力量，代表走向进步的上行轨迹
- **周期**围绕着上行轨迹反复出现，像钟摆一样在两个极端之间更迭
- 两者共同构成**螺旋式上行轨迹**，显现在一切事物中

```
进化 = 知识的积累 > 知识的遗失 → 生产率稳步提高
周期 = 围绕趋势线的波动 → 繁荣与萧条交替
整体 = 螺旋式上行（上行的螺丝锥）
```

**关键数据佐证：**
- 全球实际人均GDP：1800年前增长极其缓慢，此后加速上升
- 全球预期寿命：从1800年的约30岁提升到现在的70岁以上
- 这些上行趋势相对于围绕趋势线的波动非常明显，表明人类创造力极其强大

### 1.3 大周期的基本结构

大周期在两个时期之间产生更迭：

| 维度 | 和平与繁荣时期 | 萧条、革命与战争时期 |
|------|---------------|---------------------|
| 持续时间 | 较长（40~80年） | 较短（10~20年） |
| 时间比率 | 约占5/6 | 约占1/6 |
| 特征 | 创造力、生产力、生活水平提高 | 财富和权力斗争、大量破坏 |
| 功能 | 建设与积累 | 清除弱点和过度，打造新基础 |

**大周期的时间框架：**
- 重要帝国通常延续约**250年**（上下误差150年）
- 帝国内部的经济、债务和政治大周期持续**50~100年**
- 长期债务周期：约**100年**
- 短期债务周期：约**8年**
- 破坏/重组时期：通常**10~20年**
- 和平与繁荣时期：通常**40~80年**

## 第二部分：帝国兴衰的典型规律

### 2.1 三阶段模型

达利欧将国家兴衰分为三个阶段：

#### 上升阶段（新秩序建立后的繁荣建设时期）

**基本面特征：**
- 债务水平相对较低
- 财富、价值观和政治差距相对较小
- 人们通过有效合作创造繁荣
- 具备良好的教育和基础设施
- 拥有强有力且有能力的领导者
- 一个或多个世界大国主导和平的世界秩序

**上升的因果链条：**
```
教育实力 → 创新和技术 → 竞争力提升 → 全球贸易份额扩大
→ 军事实力增强 → 经济产出增加 → 建设世界领先金融中心
→ 建立储备货币地位
```

**具体表现：**
- 强大的教育实力（传授知识和技能，培养坚韧性格和良好修养）
- 创新和发明新科技（如荷兰造船术、资本主义发明）
- 对全球最佳思维的开放态度
- 劳动者、政府和军队之间通力合作
- 发展资本市场（贷款、债券和股票市场）
- 建设世界领先的金融中心
- 本国货币成为世界储备货币

#### 顶部阶段（成功的回报中埋下衰落的种子）

**典型过度特征：**
- 高负债
- 财富、价值观和政治差距巨大
- 教育水平和基础设施不断下滑
- 国家内部不同群体之间发生冲突
- 过度扩张的国家受到新兴对手挑战

**衰落的内在机制：**
1. **劳动力成本上升**：国家富有后，劳动者要求更高工资，竞争力下降
2. **技术扩散**：其他国家复制领先大国的方法和技术
3. **精神懈怠**：人民变得更富有后不再努力工作，追求悠闲生活
4. **债务泡沫**：人们习惯富裕生活，通过借款维持优越生活
5. **贫富差距扩大**：资本主义制度下金融收益分配不均
6. **储备货币的"过分特权"**：鼓励过度借款
7. **帝国维持成本超过收益**：变得无利可图

#### 下跌阶段（痛苦的争斗和结构性重组）

**内部表现：**
- 债务极其庞大，经济衰退
- 被迫在违约和大量印钞之间选择
- 货币贬值和通胀加剧
- 政治极端主义：左翼或右翼民粹主义抬头
- 富人被加征税收，资金外流
- 强有力的民粹主义领导人出现
- 革命或内战爆发

**外部表现：**
- 崛起大国挑战现有世界秩序
- 军事战代价极其高昂但推动结构性变革
- 持有储备货币和债务的债权人开始抛售

### 2.2 大周期结束的标志

> "当所有这些因素（负债、内战/国内革命、国外战争、对货币失去信心）交织在一起时，世界秩序的改变通常就近在咫尺。"

**关键信号：**
1. 持有储备货币的债权人对其失去信心，开始抛售
2. 国家无法用未贬值的货币来偿还累积的债务
3. 内部秩序的改变与外部挑战交织在一起

### 2.3 世界秩序更迭的典型模式

当帝国更迭时（如荷兰→英国→美国），以下大部分或所有情况都会发生：

| 维度 | 旧秩序终结 | 新秩序建立 |
|------|-----------|-----------|
| 经济 | 经济实力相对衰落 | 经济实力相对崛起 |
| 金融 | 金融中心地位衰退 | 新金融中心崛起 |
| 货币 | 储备货币地位丧失 | 新储备货币确立 |
| 军事 | 军事实力下降 | 军事实力上升 |
| 技术 | 创新能力减弱 | 技术领先地位确立 |
| 内部 | 内部冲突加剧 | 新秩序凝聚共识 |
| 国际 | 盟友关系瓦解 | 新联盟体系形成 |

## 第三部分：18个决定因素详解

达利欧构建了18个决定因素来解释国家兴衰，分为三大类别。

### 3.1 继承的决定因素（4个）

#### 因素1：地理
- **定义**：国家的位置、周围环境、地形地貌
- **核心规律**：辽阔国土和天然屏障有利于形成大一统国家
- **案例**：美国和中国均有辽阔国土和天然屏障；欧洲自然边界众多，强化了分裂格局

#### 因素2：地质
- **定义**：地表和地下的自然资源
- **核心规律**：每种大宗商品的价值（经通胀调整后）长期呈下降趋势
- **风险**：高度依赖一种或几种大宗商品是最脆弱的定位

#### 因素3：天灾
- **定义**：流行病、洪水和干旱等
- **核心规律**：影响力甚至超过战争和经济萧条
- **功能**：充当压力测试，揭示社会的潜在优势和劣势
- **历史案例**：黑死病夺去0.75亿~2亿人生命；20世纪天花夺去3亿多人生命

#### 因素4：系谱
- **定义**：人口基因构成的影响
- **核心规律**：仅约15%的行为差异可以用基因差异解释
- **地位**：相对其他因素是次要决定因素

### 3.2 人力资本的决定因素（10个）

#### 因素5：自身利益
- **核心规律**：最强大的推动力
- **关键区别**：个人利益 vs 家庭利益 vs 国家利益
- **决定性作用**：能否将集体利益置于个人利益之上是国家成功的关键

#### 因素6：获得并维持财富和权力的动力
- **核心原则**："自给自足+"——收入至少需与支出相当
- **警示**：支出一旦超过收入，就离苦难和动荡不远了
- **公式**：财富 = 购买力 ≠ 货币和信贷

#### 因素7：资本市场
- **核心规律**：储蓄和通过资本市场获得购买力的能力
- **重要性**：资本市场的发达程度是国家成功的重要决定因素
- **历史意义**：荷兰人发明了第一家公开上市公司和第一个股票市场

#### 因素8：借鉴历史的能力
- **核心规律**：大多数人不具备这种能力，仅从自己经历中学习远远不够
- **正面案例**：中国在借鉴历史方面较为擅长
- **警示**：不了解至少1900年以来的历史及其与当前形势的关联，很可能会遇到麻烦

#### 因素9：多代人心理周期
- **核心规律**："富不过三代"
- **时间对应**：三代人的时间大致相当于一个长期债务周期（约100年）
- **五阶段演变**：
  1. 人民和国家都贫困，也认为自己贫困
  2. 人民和国家都富裕了，但仍认为自己贫困（最具生产力阶段）
  3. 人民和国家都富有，也认为自己富有（健康高峰）
  4. 人民和国家变贫困了，但仍认为自己富有（早期衰落）
  5. 人民和国家都贫困，也认为自己贫困（明显衰落）

#### 因素10：将即时满足置于长远福祉之上
- **核心规律**：夸大周期的高峰和低谷
- **影响**：政府尤其容易受此影响，政客优先考虑短期利益

#### 因素11：人类的创造力
- **核心规律**：最伟大的能力是推动进化
- **公式**：创新发明 + 商业精神 + 兴旺发达的资本市场 = 生产率大幅提升 = 财富和权力增加
- **特征**：不同社会的创造力差异显著

#### 因素12：文化
- **核心原则**："文化就是命运"
- **机制**：文化差异影响人们对待彼此的方式
- **形成**：基于对现实世界的看法而形成

#### 因素13：对全球思维的开放态度
- **核心规律**：先行指标——自我孤立的实体往往错过时机
- **警示**：国家和王朝失败最常见的原因之一
- **历史案例**：中国唐末、明末和日本江户时代的孤立主义导致技术落后

#### 因素14：领导力
- **核心规律**：每一代中约几百人改变了一切
- **关键**：关键人物的独特性可以改变历史进程
- **不同阶段需要不同类型领导人**：
  - 第六/第一阶段："励志将领"（顽强机智的勇士）
  - 第二阶段："土木工程师"（设计和构建高效体制）
  - 第三阶段："鼓舞人心的远见卓识者"
  - 第四阶段："稳扎稳打、纪律严明的领导者"
  - 第五阶段："强有力的和平缔造者"或"强有力的革命者"

### 3.3 群体互动形成的决定因素（4个）

#### 因素15：贫富差距
- **核心规律**：扩大往往导致冲突加剧
- **加剧条件**：经济形势恶化时影响更为严重
- **量化指标**：当60%~80%的危险信号出现时，爆发严重冲突的可能性约1/6

#### 因素16：价值观差距
- **核心规律**：宗教和意识形态差异
- **加剧条件**：经济承压时，差距越大冲突越大

#### 因素17：群体斗争
- **核心规律**：权力通常由三四个群体掌控
- **最常见的划分**：富人 vs 穷人
- **周期特征**：大周期早期和谐，后期争斗更多
- **历史规律**：那些最广泛地使用人才、根据个人业绩而非特权赋予公民责任的社会最可能取得可持续性成功

#### 因素18：左倾/右倾政治周期
- **周期**：约10年
- **机制**：经济繁荣期→贫富差距扩大→政治极化
- **警示**：重大经济危机预示革命到来

### 3.4 额外重要因素（扩展因素）

达利欧实际论述了22个影响因素，除上述18个外还包括：

- **因素19：囚徒困境**——要想维系和平，就必须解决囚徒困境
- **因素20：双赢/双输关系**——取决于双方做出的选择
- **因素21：权力平衡大周期**——驱动国家内部和国家之间的和平/战争大周期
- **因素22：军事实力与和平/战争周期**——决定最终结果的关键因素

### 3.5 五大力量

达利欧特别强调**5个最重要的因素**（"五大力量"）：

1. **有利和不利的金融周期**（资本市场周期）
2. **内部秩序和混乱周期**（合作程度与财富和权力斗争）
3. **外部秩序和混乱周期**（现有大国在财富和权力斗争中的竞争力）
4. **创新和技术发展步伐**（解决问题和做出改进）
5. **天灾**（干旱、洪水和疾病）

> "当5个决定因素朝着同一个方向出现改善或恶化时，大多数其他因素都会出现相同的走势。"

## 第四部分：内部秩序周期的6个阶段

### 4.1 六阶段模型详解

#### 第一阶段：新秩序开始，新领导层巩固权力
- **特征**：获胜者夺取控制权，清除残余反对派
- **关键行动**："清洗"——消除前领导者的忠诚分子，胜者争夺权力内斗
- **代表领导者**：唐太宗、罗马皇帝奥古斯都、美国国父、拿破仑、俾斯麦

#### 第二阶段：资源配置体系与政府官僚机构建立和完善
- **特征**：早期繁荣时期，和平与繁荣时期的开始
- **核心原则**：一个体制要想成功，就要为大多数人（特别是广大中产阶级）创造繁荣
- **代表领导者**：康拉德·阿登纳、李光耀、邓小平
- **关键**：设计和创建一个对大多数人来说都高效的体制

#### 第三阶段：出现和平与繁荣
- **特征**：中期繁荣时期，周期中的最佳时期
- **表现**：
  - 近乎平等的教育机会和择优录用的工作岗位
  - 成功的企业家、发明家和冒险家成为英雄
  - 债务增长促进生产率提高
  - 收入超过支出，储蓄超过负债
- **代表时期**：英国维多利亚时代、19世纪末德意志帝国、20世纪60年代美国

#### 第四阶段：过度时期（泡沫繁荣时期）
- **典型特征**：
  - 债务融资购买活动迅速增多
  - 更多资金用于消费和奢侈品，更少用于生产性投资
  - 军事建设耗资巨大
  - 国际收支状况恶化
  - 财富和机会差距巨大
- **历史案例**：1988-1990年日本、1929年美国、2006-2007年美国

#### 第五阶段：财政状况糟糕，冲突激烈
- **典型有害组合**：
  1. 国家及其民众处于糟糕的财务状态
  2. 实体内部的收入、财富和价值观存在巨大差距
  3. 受到严重的负面经济冲击
- **伴随问题**：堕落、官僚主义、民粹主义和极端主义、族群斗争、公共领域真相消失、规则不被遵守
- **当前状态**：美国正处于第五阶段（达利欧判断）

#### 第六阶段：出现内战/革命
- **特征**：极其残酷的决死之战
- **规律**：内战/革命注定会发生，从根本上改变内部秩序
- **后果**：财富和政治权力的彻底重组，包括债务、财务所有权和政治决策权
- **持续时间**：历史学家所标记的内战时期通常持续数年

### 4.2 当前美国的位置判断

**达利欧的核心判断：美国正处于第五阶段**

**关键指标：**
- 内部冲突指数处于20世纪20年代初以来最高水平
- 贫富差距巨大（顶层0.1%人口净值≈底层90%总和）
- 债务负担高（非金融债务GDP的227%）
- 政治冲突激烈
- 约60%~80%的经济危险信号已出现

**概率估算：**
- 爆发严重内部冲突的可能性约1/6
- 未来10年陷入第六阶段（类似内战）的可能性约30%

## 第五部分：货币、信贷与债务大周期

### 5.1 长期债务周期6个阶段

| 阶段 | 特征 | 货币类型 |
|------|------|---------|
| 第一阶段 | 债务很少，使用硬通货 | 硬通货（金属货币） |
| 第二阶段 | 出现硬通货债权票据（纸币） | 挂钩货币体系 |
| 第三阶段 | 债务增加，债权票据超过硬通货 | 第二类货币体系 |
| 第四阶段 | 债务危机、违约、与硬通货脱钩 | 向法定货币过渡 |
| 第五阶段 | 法定货币，最终导致货币贬值 | 法定货币体系 |
| 第六阶段 | 回归硬通货 | 重建信心 |

### 5.2 三种货币体系

| 类型 | 特征 | 约束程度 | 历史案例 |
|------|------|---------|---------|
| 第一类：硬通货 | 金属硬币 | 最具约束性 | 古代社会 |
| 第二类：纸币 | 硬通货债权 | 中等约束 | 金本位制 |
| 第三类 | 法定货币 | 无约束 | 1971年后的美元体系 |

### 5.3 债务危机处理的4种工具

1. **财政紧缩**（减少支出）——具有通缩性，痛苦且不可持续
2. **债务违约和重组**——具有通缩性，对双方都痛苦
3. **财富转移**（对富人增税）——政治上有挑战性
4. **印钞并使货币贬值**——最简便、最常见、最不被了解的手段

> "印钞是最简便、最不被人了解、最常见的主要债务重组手段。"

### 5.4 货币贬值的历史教训

**1700年以来的750种货币：**
- 仅约20%目前仍然存在
- 所有现存货币均出现过贬值
- 1912年以来，现金货币的实际回报率是-0.1%
- 黄金在同期的实际回报率是1.6%

**主要货币的6次大幅贬值：**
1. 19世纪60年代（美国内战）
2. 一战期间及战后
3. 20世纪30年代大萧条
4. 二战期间
5. 1968-1973年（布雷顿森林体系崩溃）
6. 2000年以来的渐进贬值

## 第六部分：外部秩序与战争周期

### 6.1 国家间斗争的5种类别

| 类别 | 内容 | 升级路径 |
|------|------|---------|
| 1. 贸易/经济战 | 关税、进出口限制 | → |
| 2. 技术战 | 技术共享与国家安全 | → |
| 3. 地缘政治战 | 领土和联盟 | → |
| 4. 资本战 | 经济制裁、限制资本市场准入 | → |
| 5. 军事战 | 实际动武和军事力量部署 | 终极形式 |

> "真枪实弹的战争爆发之前，通常会有一场经济战。在宣布全面战争之前，双方会有大约10年的经济、技术、地缘政治和资本战争。"

### 6.2 战争爆发的关键条件

**军事战争风险最大的两种情况：**
1. 双方的军事实力旗鼓相当
2. 双方在生存问题上存在不可调和的分歧

**关于战争的两个确信点：**
1. 战争不会按计划进行
2. 战争远比想象的更糟糕

### 6.3 权力平衡的5个步骤

1. **联盟的形成**——"敌人的敌人就是朋友"
2. **战争决定胜负**——旗鼓相当才会激烈交战
3. **胜者之间的斗争**——"清洗"
4. **步入和平与繁荣**——但最终变得过度扩张
5. **不断激化的冲突**——导致国内和国际秩序巨变

## 第七部分：中国在大周期中的位置

### 7.1 中国历史的大周期视角

**中国在历史上的地位：**
- 1500年以前，中国几乎一直是最强大的国家
- 明朝是当时世界上最先进、最强大的帝国
- 人口1亿~1.3亿（欧洲仅5000万~5500万）
- 拥有全世界规模最大的海军和100万人的常备军
- 识字率极高，医学先进，金融体系相对发达

**中国的朝代周期：**
- 自公元600年（唐朝）以来，中国经历了多个完整的兴衰大周期
- 每个朝代大致经历：兴起→繁荣→过度→衰落→革命/战争→新秩序
- 达利欧研究了中国自唐朝以来的所有主要朝代，发现遵循与欧洲帝国相似的周期规律

**中国的特殊性：**
- 中国在借鉴历史方面较为擅长
- 中国经历过几次复兴（罗马人和希腊人从未东山再起）
- 中国特色社会主义是从各种体制演化而来的混合体

### 7.2 当前中国在大周期中的位置

**总体判断：中国处于上升阶段，正在迅速崛起**

#### 优势领域

| 维度 | 具体表现 |
|------|---------|
| 经济体量 | PPP调整后占全球22%，全球最大 |
| 贸易 | 全球最大出口国，占全球出口14% |
| 教育 | 学士学位人数占全球22%，排名第二 |
| 创新 | 研发支出、专利、科研人员全球占比大，迅速崛起 |
| 基础设施 | 生产性投资占全球比例最高 |
| 经济增长 | 未来10年预期实际增长率年均4.3% |
| 军事 | 实力迅速上升 |
| 文化 | 民众自给自足性和工作勤劳度高 |

#### 劣势领域

| 维度 | 具体表现 |
|------|---------|
| 储备货币 | 人民币国际化仍在初期 |
| 法治/腐败 | 相对较弱 |
| 贫富差距 | 收入最高1%占14%、最高10%占41% |
| 债务 | 非金融债务水平高（GDP的263%），但96%以本币计价 |
| 自然资源 | 更依赖外部 |
| 政府债务 | 相对较低（GDP的48%） |

#### 多代人心理周期评估

根据达利欧的五阶段模型，中国目前处于**第二阶段向第三阶段过渡**：
- 第二阶段（后期新兴国家）：人民富裕了但仍认为自己贫困，高储蓄率，大量投资于生产性活动
- 正在进入第三阶段（健康高峰国家）：收入高，劳动力变得更昂贵，但前期投资仍在带来回报

### 7.3 中美关系与冲突评估

#### 当前状态
- 中美冲突规模比正常水平高出一个标准差
- 正处于4类战争中：贸易/经济战、技术战、资本战、地缘政治战
- 尚未发生第五类战争（军事战），但风险在加大
- 从4类战争发展到军事战通常需要5~10年

#### 达利欧的核心判断

1. **美国正在衰落，中国正在崛起**——这一趋势非常清晰
2. 在可预见的未来，中美将有足够实力对彼此造成不可接受的伤害
3. "相互保证毁灭"的前景应能防止大规模军事战
4. 未来10年爆发大规模战争的概率约**35%**
5. 最大的风险触发点：**台湾问题**
6. "打仗的最佳方式是不战而屈人之兵"

#### 关键时间窗口
- 下一个大风险点约在**5年之后**出现（误差两三年）
- 各国联盟关系可能在约5年内变得更加牢固
- 中国正在不断赢得盟友，美国正在失去盟友

#### 达利欧的建议

> "如果一方的相对实力正在下降，那么尽早开战为好；如果相对实力正在上升，那么推迟战争更合适。"

**对中国而言：** 由于相对实力正在上升，推迟冲突、继续发展更为有利。

**对双方而言：** 进行良好的协商，既考虑到对方也考虑到自身的优先关注点，妥善在二者之间进行权衡，寻求双赢结果。

## 第八部分：投资启示与应对原则

### 8.1 从大周期角度思考投资

**市场回报的4个决定因素：**
1. 经济增长
2. 通胀
3. 风险溢价
4. 折现率

**历史教训：**
- 1900年以来的10个大国中，有7个国家至少经历过一次几乎所有财富全部消失
- 即使是战胜国的投资者也面临市场时机和税收两大不利因素
- 在大势不好时，大多数投资者在股价接近低点时卖出

### 8.2 核心投资原则

1. **分散化投资**高于一切
2. **黄金**在60/40投资组合下跌时通常表现良好
3. 关注你得到的**利息金额是否足以弥补货币贬值风险**
4. 当**金融资产大大超过实物资产**时，走势可能逆转
5. 在**长期债务周期后期**，持有债务如同手握定时炸弹

### 8.3 应对大周期的原则总结

| 序号 | 原则 |
|------|------|
| 1 | 没有任何政治体制、经济体系、货币或帝国可以永远存在 |
| 2 | 从一个极端到另一个极端的变化是常态，而不是例外 |
| 3 | 和平与繁荣时期比萧条/革命/战争时期长得多（约5:1） |
| 4 | 人们往往过分关注现有情况，而忽视改变情况的永恒普适力量 |
| 5 | 人力资本是最可持续的资本 |
| 6 | 通常谁赢得技术战，谁就赢得经济战和军事战 |
| 7 | 在市场上和生活中，要想成功就应当下注于积极面，同时管理周期和颠簸 |
| 8 | 债务吞噬资产净值，但央行可以通过印钞来为债务融资 |
| 9 | 拥有储备货币是一个国家所能拥有的最大权力之一 |
| 10 | 所有货币都会贬值甚至消亡 |
| 11 | 当解决分歧的体制变得无效时，内战开始 |
| 12 | 若你有疑虑，就要撤离 |
| 13 | 获胜意味着在不失去最重要的东西的前提下，得到最重要的东西 |
