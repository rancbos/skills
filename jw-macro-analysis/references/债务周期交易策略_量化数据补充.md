### 2.2 货币条件判定：DR007月均值

DR007（银行间存款类机构7天质押式回购利率）是央行货币政策操作目标之一，定量反映金融体系流动性供需。

- **DR007持续低于政策利率** → 货币条件宽松 → 利多债市
- **DR007持续高于政策利率** → 货币条件收紧 → 利空债市

**最新数据（2025年）**：
- DR001年加权平均利率：**1.46%**（较2024年下降19bp）
- DR007年加权平均利率：**1.63%**（较2024年下降19bp）
- 央行7天逆回购操作利率（2025年）：**1.50%**
- 2026年5月末：DR007约**1.39%**，低于央行1.40%政策利率

## 三、周期拐点交易信号量化

### 3.1 核心领先指标相关系数（2018年以来，vs 10年期国债收益率）

| 指标 | 相关系数 | 方向 | 领先性 | 量化结论 |
|------|----------|------|--------|----------|
| **社融-M2同比增速差** | **0.70** | 正相关 | 领先 | **最强信号**，增速差扩大→利率抬升 |
| M2-M1同比增速差 | **-0.53** | 负相关 | 领先 | 企业活跃度指标，差值收窄→利率下行 |
| 社融同比增速 | **0.25** | 正相关 | 同步略领先 | 宽信用效果直观指标 |
| 企业中长贷占比 | **-0.14** | 负相关 | 指引较差 | 受房地产信贷扰动大 |

> 数据来源：书中第一章，万得资讯回溯分析。社融-M2增速差是利率走势的最强先行指标。

**传导逻辑**：
- 社融-M2增速差走阔 → 实体融资需求快于M2派生 → 货币需求旺盛 → 资金价格上升 → 利率上行
- M2-M1增速差收窄 → 企业存款活期化 → 经营预期乐观 → 利率上行（负相关）

### 3.2 资金面信号阈值

| 信号 | 阈值 | 含义 | 交易含义 |
|------|------|------|----------|
| DR007 vs 政策利率 | 持续低于 | 货币条件宽松 | 利多债市 |
| DR007 vs 政策利率 | 持续高于 | 货币条件收紧 | 利空债市 |
| **隔夜回购占比** | **>87%** | 杠杆过高预警 | 关注去杠杆风险 |
| **隔夜回购占比** | **>90%** | 极端杠杆 | 去杠杆风险极高 |
| 隔夜回购占比 | 正常70%+ | 市场正常 | — |
| 超储率下降 | 边际下降 | 银行可用资金减少 | 流动性边际收紧 |

> 数据来源：书中第一章，图1-11。隔夜质押式回购成交量在所有期限回购中占比最大，通常在70%以上。87%和90%是两个重要水位线。

### 3.3 票据利率信号

| 信号 | 含义 | 交易含义 |
|------|------|----------|
| 月末票据利率较全月均值**下行** | 银行月末借助票据给信贷"冲量"，信贷弱 | **利多债市** |
| 月末票据利率较全月均值**上行** | 信贷投放景气度高 | 利空债市 |

> 改良指标：月末票据利率较全月均值的下滑幅度（书中图1-19）。2022年下半年起，单纯月末票据利率的指示性作用有所减弱。

### 3.4 期限利差信号

**核心发现**：利差拐点往往早于利率拐点出现（书中附图1）。

- 长端利率（10年期国债）：与经济基本面相关性更高，反映市场风险偏好
- 短端利率（1年期国债）：与资金面、政策面相关性更高

**收益率曲线形态与周期**：
| 形态 | 含义 | 典型阶段 |
|------|------|----------|
| 牛平 | 长端下行快于短端 | 牛市初期 |
| 牛陡 | 短端下行快于长端 | 宽货币确认 |
| 熊平 | 长端上行快于短端 | 熊市初期 |
| 熊陡 | 短端上行快于长端 | 紧货币确认 |

### 3.5 隐含税率信号（债市情绪指标）

**定义**：隐含税率 =（国开债利率 - 国债利率）/ 国开债利率

**核心特征**："牛市收窄、熊市走阔"（牛窄熊阔）

| 信号 | 条件 | 历史准确率 | 交易含义 |
|------|------|-----------|----------|
| **熊转牛拐点** | 隐含税率上破均值+2σ | 2/2次 = **100%** | 大级别"熊转牛"信号 |
| 熊转牛（弱信号） | 隐含税率上破均值+1σ | 4/7次 ≈ **57%** | 中级别信号 |
| **牛转熊拐点** | 隐含税率下破均值-2σ | 1/2次 = **50%** | 大级别"牛转熊"信号 |
| 牛转熊（弱信号） | 隐含税率下破均值-1σ | 4/6次 ≈ **67%** | 中级别信号 |

> 数据来源：书中附录，2006年以来10年期隐含税率数据。

**拐点时序关系**：
- "熊转牛"行情：隐含税率拐点**先于**长债利率拐点（因交易型机构更敏感）
- "牛转熊"行情：隐含税率拐点与长债利率拐点**同步或略滞后**

### 3.6 股债跷跷板信号

- 股债跷跷板效应在中国市场显著
- 全球风险偏好扭转时，股市拐点通常**略早于**债市
- **2018年案例**：沪深300于1月25日见顶，标普500于1月26日见顶
- 2018年Q4：全球股市大跌 → 股债跷跷板 → 债市走牛

## 四、2018年实战案例量化数据

2018年是"宽货币+紧信用"的经典案例，10年期国债收益率全年走势：

| 阶段 | 时间 | 利率区间 | 关键事件 |
|------|------|----------|----------|
| 阶段一 | 1月上旬 | 上行至**3.98%** | 监管集中落地+通胀预期+股债跷跷板 |
| 阶段二 | 2-6月 | 下行至**3.48%** | 宽货币确认+贸易战+紧信用 |
| 阶段三 | 7-9月 | 回升至**3.61%** | 地方债供给+通胀升温+稳汇率 |
| 阶段四 | 10-12月 | 降至**3.23%** | 全球风险偏好下行+中美货币周期背离 |

**关键利率点位**：
- 年内高点：3.98%（1月中旬）
- 年内低点：3.23%（12月31日）
- 年内最大下行幅度：75bp
- 7月23日国务院常务会议前低点：3.44%

**关键教训**：宽货币向宽信用传导需要宽财政配合，否则"推绳子"效应明显。

## 六、利率定价锚定关系

### 6.1 MLF利率与10年期国债

- 10年期国债收益率以**1年期MLF利率**为中枢
- 超过**70%**的时间内，10年期国债围绕MLF利率上下波动**30bp**
- 偏离MLF利率的时段均存在强外部因素干扰

### 6.2 LPR传导机制

```
MLF利率 → LPR → 贷款利率
```

- 2019年LPR改革后，LPR由MLF利率加点形成
- 贷款利率定价：LPR + 银行加点
- 债券利率市场化程度显著高于信贷市场

## 七、流动性缺口模型（量化公式）

### 7.1 银行间流动性缺口

```
银行间流动性缺口 = Δ法定存款准备金需求
                 + Δ超额存款准备金需求
                 + Δ货币发行
                 - Δ央行公开市场操作净投放
                 - Δ外汇占款
                 - Δ财政存款
```

### 7.2 基础货币变动

```
Δ基础货币 ≈ Δ外汇占款 + Δ央行流动性净投放 - Δ政府存款
```

### 7.3 超储率测算

```
超储率 = [其他存款性公司储备资产 - (M2-M0-各类保证金存款) × 平均存款准备金率] ÷ 各项存款余额
```

### 7.4 银行间市场杠杆率

```
杠杆率 = 银行间债券市场托管量 ÷ (银行间债券市场托管量 - 待购回债券余额)
```

### 7.5 空转倍数

```
空转倍数 = (表内外信贷 + 未到期商业汇票 + 理财产品存续余额 + 同业融入)
         ÷ (扣除票据的信贷 + 企业债券 + 信托贷款 + 委托贷款 + 商业汇票)
```

## 八、金融空转治理规律

### 8.1 两轮典型空转

| 轮次 | 时间 | 模式 | 监管应对 |
|------|------|------|----------|
| 第一轮 | 2015-2016 | "自营—货基—同业存单—同业理财"空转链 | 2017年MPA考核+同业存单纳入同业负债 |
| 第二轮 | 2019-2020 | "票据—结构性存款"套利链 | 限制结构性存款规模和收益率 |

### 8.2 治理规律

**金融监管收紧力度 ∝ 空转复杂程度 / 基本面状况**

- 基本面好 → 监管严格（如2015-2016年）
- 基本面差 → 监管温和（如2020年）

## 十、实操检查清单（量化版）

### 日常监控
- [ ] DR007月均值 vs 央行7天逆回购利率（偏离方向和幅度）
- [ ] 社融-M2同比增速差（核心信号，相关性0.70）
- [ ] M2-M1同比增速差（企业活跃度，相关性-0.53）
- [ ] 隔夜回购占比（预警线87%/90%）
- [ ] 超储率变化（官方公布 vs 测算值）
- [ ] 月末票据利率较全月均值变化（下行=信贷弱=利多债市）
- [ ] 隐含税率及其历史分位（±1σ/±2σ信号）

### 周期拐点判断
- [ ] 社融增速拐点确认（连续2-3个月同向变化）
- [ ] 央行货币政策措辞变化（"灵活适度"→"中性"→"略紧"）
- [ ] OMO/MLF操作量变化（超额续作→等额→缩量）
- [ ] 期限利差拐点（通常早于利率拐点）
- [ ] 隐含税率极端值信号（±2σ历史分位）
- [ ] 股债跷跷板方向变化（股市拐点略早于债市）

### 风险预警
- [ ] 理财赎回压力（债市快速转熊的放大器）
- [ ] 地方债供给冲击
- [ ] 人民币汇率贬值压力（限制宽货币空间）
- [ ] 信用风险事件（民企违约、城投暴雷）
- [ ] 金融空转监管收紧
