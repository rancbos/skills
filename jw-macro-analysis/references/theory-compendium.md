## 一、四周期嵌套模型（周金涛康波理论）

经济周期由四层嵌套：康波(50-60年) > 房地产(18-20年) > 朱格拉(9-10年) > 基钦(3-4年)。

**锁模效应**：各周期转折点趋于一致——每2个存货周期低谷与1个资本支出周期低谷重合，每4个存货周期低谷与1个房地产周期低谷重合。

**康波四阶段**：繁荣(高增长低通胀→股票) → 衰退(增速降通胀高→商品→现金) → 萧条(低增长通缩→黄金/债券) → 回升(增速恢复通缩→股票)

**第五次康波定位(1991至今)**：1991-2004繁荣(IT驱动) → 2004-2008衰退起点 → 2008-2018衰退(两次冲击) → 2018-至今萧条期

**详细报告**：周金涛康波理论深度分析报告.md、周金涛理论大集_优化补充.md、人生财富靠康波_量化数据补充.md

## 二、债务周期理论（达利欧模型）

**通缩性债务周期7阶段**：早期(债务≤收入) → 泡沫(债务/GDP年增20-25%) → 顶部(信贷问题早于经济峰值半年) → 萧条(股价跌50%、GDP降10%) → 和谐去杠杆(年印钞GDP的4%) → 推绳子(QE衰减) → 正常化(5-10年)

**泡沫识别7特征**：资产价格偏高、预期继续涨、普遍看涨、高杠杆、买家囤积、新买家涌入、刺激性货币政策

**详细报告**：达利欧深度研究报告_债务周期与大周期理论_深化补充版.md、债务危机_量化数据补充.md

## 六、通胀分析框架（福布斯）

**两种通胀**：非货币性(供需短期可矫正) vs 货币性(货币贬值持续性强)

**判断方法**：金价持续数月上涨 + CPI/大宗商品/汇率同步 → "百分之百确定"是货币性通胀

**6级预警体系**：一级黄金(日频领先数月) → 二级大宗(滞后黄金1年) → 三级M0/M2(月度) → 四级美联储政策 → 五级政府财政 → 六级消费者行为

**沃尔克标准**：金价涨→收紧货币，金价跌→扩大货币

## 九、塔勒布风险理论体系

**肥尾分布**：κ统计量衡量肥尾程度，VaR在肥尾下严重低估风险。三层尾部：薄尾(指数)、中等肥尾(幂律α>3)、极端肥尾(α≤3)

**反脆弱**：三元结构(脆弱-强韧-反脆弱)、凸性效应(损失有限收益无限)、林迪效应(存活越久预期寿命越长)、否定法

**非对称风险**：银律>金律、决策者必须承担后果、代理人问题

**杠铃策略**：85-90%极安全 + 10-15%极冒险。活着最重要。

## 十二、认知偏差与行为经济学（卡尼曼）

**双系统**：系统1(快/自动/易偏误) vs 系统2(慢/费力/较准确)

**14种关键偏差**（对宏观分析最相关的）：
- 过度自信：CEO相关系数仅0.30→导致泡沫
- 锚定效应：锚定指数55%→影响价格预期
- 可得性偏差：恐惧/贪婪放大器
- 损失厌恶：系数2:1→波动加剧
- 框架效应：同一信息不同表述→不同决策

**前景理论**：价值函数在损失区域更陡(损失厌恶)、确定性效应、参考点依赖

## 十五、微观经济学基础（马歇尔）

**马歇尔剪刀**：价值由需求和供给共同决定，如剪刀两刃

**弹性理论**：价格变动1%引起需求量变动的百分比。格雷戈里·金法则：供给减10%→价格涨30%

**准租金**：旧投资收入短期不受成本影响，长期趋向正常利息——解释短期价格粘性

**信心-信用循环**（行3150）："有信心就会令信心更足；信用会增加购买手段，使物价得到些许回升……周而复始。"
