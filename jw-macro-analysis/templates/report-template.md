# 中国周期方向判断报告模板

> 每次分析全量输出完整报告。

**必填字段**（报告必须包含，不可跳过）：
- 一、核心结论（位置+风险等级+关键信号+综合判断）
- 十、关键数据表（至少8个指标）

**选填字段**（数据缺失时可标注"数据暂缺"）：
- 六、中国特色分析（4.5 信任变量可选）
- 九、走势特征（仅供参考，不构成投资建议）

**数值格式规范**：
- GDP增速/CPI/PPI/PMI/M2增速/LPR：保留1位小数，单位%
- 债务/GDP：保留整数，单位%
- 房价/收入比：保留1位小数
- 汇率：保留4位小数
- 金价：保留2位小数，单位美元/盎司
- 社融/信贷：保留整数，单位亿元
- 所有同比增速：带+/-号，如 "+5.2%"、"-3.1%"
