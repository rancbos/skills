### Task 2.1: 在 pipeline_phase2.py 中添加 candidate_count 检查

def apply_safety_valve(scores: dict, max_total: int = 100, max_recommended: int = 30):
    """
    Safety valve: prevent token-wasting when too many candidates.
    Returns (modified_plan, valve_triggered: bool)
    """
    total = scores.get("candidate_count", 0)
    if total <= max_total:
        return None, False

    plan = build_summary_plan(clusters, scores)
    # Trim: keep top-30 by score, rest → appendix
    recommended = plan.get("recommended", [])
    if len(recommended) > max_recommended:
        plan["recommended"] = recommended[:max_recommended]
        plan["appendix"] = plan.get("appendix", []) + recommended[max_recommended:]
        plan["valve_triggered"] = True
        plan["valve_reason"] = f"candidate_count={total} > {max_total}, trimmed to top-{max_recommended}"
    return plan, True
```

**Step 2: 集成到 run_pipeline()**

在 build_summary_plan 调用后插入安全阀检查。

```
