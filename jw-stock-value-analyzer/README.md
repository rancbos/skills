# jw-stock-value-analyzer

股票价值分析器（Stock Value Analyzer）v1.8

## 简介

基于邱国鹭《投资中最简单的事》方法论的股票价值分析器。v1.8 三档输出(brief/standard/deep)+统一图标+可视化评分卡+一句话摘要+能力边界声明+标准化来源块。v1.7 红线扫描+辩论+决策日志。
