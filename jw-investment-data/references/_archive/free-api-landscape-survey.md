# 免费金融数据 API 全景调研 & 落选说明

本文档记录 jw-investment-data 在选型过程中考察过的全部数据源，包括**入选源**（已纳入 skill 的主力/备选引擎）和**落选源**（考察后决定不纳入的源及原因）。

## 入选源

| 数据源 | 入选原因 | 在 skill 中的角色 |
|--------|---------|-----------------|
| **AkShare** | 面最广（A/港/美/期货/基金/宏观），开源免费，维护活跃 | 主引擎（A股） |
| **efinance** | 基于东方财富，轻量即用，接口稳定 | 备选引擎（A股） |
| **yfinance** | Yahoo 数据，港美股最全，info 字段丰富 | 主引擎（港股/美股） |
| **Baostock** | 完全免费 0 注册，A股历史数据稳定 | 第三校验源 |
| **腾讯 qt.gtimg.cn** | HTTP 直连，无依赖，纯文本解析 | HTTP 回退源 |
| **雪球 xueqiu.com** | 网页抓取可获取完整行情+估值数据 | HTTP 回退源 |
| **Jin10 MCP** | 实时快讯/行情，已接入 | 外汇第三方校验 |
| **pandas-datareader** | 学术标准，Fred/OECD 宏观数据 | 宏观第二源 |

## 落选源（及原因）

### Tushare Pro
- **入选理由**：国内最早一批财经数据接口，数据质量好
- **落选原因**：
  1. 需要注册实名认证（身份证+手机号）
  2. 免费版日调用仅 200 次，稍高频率立即超限
  3. 积分体系复杂，高级数据需充值
  4. AkShare + Baostock 已覆盖其免费层 90% 功能
- **结论**：门槛太高，不适合作为"即装即用"引擎

### xtquant（迅投 QMT）
- **入选理由**：迅投 QMT 官方 Python 接口，行情+交易一体
- **落选原因**：
  1. 必须安装并启动 MiniQMT 客户端（GUI 程序）
  2. 仅支持 Python 3.6-3.12，版本约束严格
  3. 部分功能需要实盘账户
  4. 不适合纯数据获取场景（太重）
- **结论**：适合已开 QMT 账户的实盘交易用户，不适合通用数据 skill

### 掘金量化 (gm)
- **入选理由**：聚合理财数据+策略研究+交易，SDK 完善
- **落选原因**：
  1. 需要注册 token，免费层限制严格
  2. 不开客户端无法取数
  3. 数据覆盖不如 AkShare
- **结论**：有账号的人可以用，不适合作为默认引擎

### 聚宽 (jqdatasdk / JoinQuant)
- **入选理由**：数据质量好，SDK 规范
- **落选原因**：
  1. 免费试用仅 1 年，之后必须付费
  2. API 需要 token 注册
  3. 不开终端无法取到实时数据
- **结论**：商业化产品，不适合长期免费使用

### 东方财富 Choice
- **入选理由**：机构级数据，覆盖极广
- **落选原因**：**纯付费产品**，无免费套餐，机构客户年费数万起
- **结论**：超出免费 skill 的范围

### 同花顺 iFinD
- **入选理由**：与 Wind 齐名的机构级终端
- **落选原因**：同 Choice，纯付费，个人版亦不便宜
- **结论**：超出免费 skill 的范围

### iTick
- **入选理由**：永久免费套餐，REST+WebSocket，实时行情强
- **未作为主引擎原因**：
  1. 需要注册 API Key（非零门槛）
  2. 免费层有频率限制
  3. AkShare + efinance 已能覆盖其主要功能
- **结论**：作为"备选未必须安装"引擎保留，在 references 中有速查指南

### Finnhub / marketstack
- **入选理由**：有免费层，美股数据丰富
- **落选原因**：
  1. 核心市场在美国，中国 A 股几乎无覆盖
  2. 免费层限制严格（Finnhub 60次/分钟，marketstack 100次/月）
  3. yfinance 在美股已足够好
- **结论**：仅美股的补充，不是必需品

### MOOTDX（通达信封装）
- **入选理由**：免费通达信数据接口
- **落选原因**：
  1. 小众项目，维护不稳定
  2. 依赖通达信客户端数据目录
  3. 与 AkShare 功能重叠但质量不如
- **结论**：特定场景可用，但不够通用

## 选型标准总结

| 标准 | 权重 | 说明 |
|------|------|------|
| 完全免费（无注册/无 token） | P0 | 即装即用，零门槛 |
| 覆盖 A 股 | P0 | 主要市场 |
| Python `pip install` 一条命令 | P1 | 降低维护成本 |
| 接口稳定、维护活跃 | P1 | 避免可用性断崖 |
| 有 HTTP 回退路径 | P1 | 网络受限环境兜底 |

> 🏆 **最终选型：AkShare（广度）+ efinance（轻量）+ yfinance（港美）+ Baostock（老牌）+ 腾讯/雪球（HTTP回退）**
