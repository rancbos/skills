# jw-content-summary



## 简介

将一本书或长文蒸馏为可实操的方法论单元（R/I/A1/A2/E/P/B 七段），含压力测试和关联方法论。适用于"帮我总结这本书""帮我看看这本书讲了啥""拆解方法论""提炼核心框架""summarize this book""这本书值不值得读""这篇文章的核心观点""帮我拆解一下这篇文章""拆解研报""深度阅读""distill the methodology""帮我写读书笔记""读后感""这本书...
