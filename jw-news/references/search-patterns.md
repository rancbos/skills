# 搜索模式参考

jw-news 必搜来源的实际搜索模式（v2.8.0 更新）。

## 搜索工具优先级

| 优先级 | 工具 | 说明 |
|--------|------|------|
| 1 | `mcp_minimax_search_web_search` | 中文效果最佳，首选 |
| 2 | `mcp_tavily_tavily_search` | 备用，英文强中文弱 |
| 3 | `mcp_ddg_mcp_search` | 最后手段，中文返回空概率高 |

> 不要使用 `web-search` 命令行工具或 `site:domain.com` 语法——中文环境下命中率极低。

## 必搜来源搜索词

搜索时用 `{今天日期}` 动态替换，格式 `YYYY年M月D日`（如 `2026年5月30日`）。不要写死日期。

| 来源 | 搜索词模板 | 工具 |
|------|-----------|------|
| 新华社 | `新华社 今日新闻 {今天日期}` | minimax_search |
| 人民日报 | `人民日报 今日要闻 {今天日期}` | minimax_search |
| 参考消息 | `参考消息 今日新闻 {今天日期}` | minimax_search |
| 经济日报 | `经济日报 今日新闻 {今天日期}` | minimax_search |
| 金十数据 | `mcp_jin10_list_flash` | MCP 直接调用 |

## 搜索技巧

- 日期格式：动态生成当天日期，格式 `YYYY年M月D日`
- 3-5 个关键词效果最佳
- 同一事件最多 2 个来源交叉印证
- minimax_search 中文效果最佳，优先使用
- 不要用 `site:domain.com` 语法，中文搜索引擎不支持

## 已知问题

| 问题 | 解决方案 |
|------|----------|
| DuckDuckGo 返回空 | 改用 `mcp_minimax_search_web_search` |
| 华尔街见闻 API 返回空 | 已从必搜来源移除，改用搜索引擎 |
| 百度搜索触发安全验证 | 用 Bing 替代（web-access CDP） |
