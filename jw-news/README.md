# jw-news

jw-news：最新新闻搜索 + 金十数据

## 简介

从权威中文媒体、求是网和金十数据查询最新新闻。信息来源：新华社、参考消息、人民日报、求是网、光明日报、经济日报、中国青年报、环球时报、解放军报、金十数据。金十数据：A股、港股、外汇、基金、债券、期货、美股等财经数据。
