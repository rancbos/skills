# 阶段 1 — 整书理解 (Adler 分析阅读)

## 目标

在动手提炼之前，先**真正读懂这本书**。用艾德勒《如何阅读一本书》的分析阅读法，快速完成对全书的结构化理解。

---

## 四步完成整书理解

### 步骤 1 — 结构 (Structural)

回答**四个阅读问题**（Adler 四问）:

| 问题 | 具体要做什么 |
|------|------------|
| **1. 这本书在谈什么?** | 依照书的种类与主题来分类。作者如何依次发展主题？ |
| **2. 作者说了什么，怎么说的？** | 找出主要想法/声明/论点。用最简短文字说明全书主旨。 |
| **3. 这本书有道理吗？** | 是全部还是部分有道理？做出自己的判断。 |
| **4. 这本书与自己的关系？** | 这本书提供的资讯有什么意义？为什么作者认为重要？ |

同时完成:
- **书的类型**（方法论/传记/哲学/实操手册/散文/学术/...）
- **主旨一句话**: 必须真的能压缩到一句
- **主要部分关系**: 列出3–7个一级论点，标出关系（并列/递进/对比/反驳）
- **作者想解决的核心问题**

### 步骤 2 — 解释 (Interpretive)

- **关键术语**: 列出作者反复使用、有特定含义的概念词，给每个词写一句"作者本人的用法"定义
- **核心命题**: 用你自己的话重述作者的5–15个核心主张
- **论证链**: 这些主张之间是怎么推导的？作者用什么证据支撑？

### 步骤 3 — 批判 (Critical) ★ 最容易被跳过也最重要

**评论前的智慧礼节规则:**
- 除非已经完成大纲架构、也能诠释整本书了，否则不要轻易批评
- 不要争强好胜，非辩到底不可
- 要能区别真正的知识与个人观点的不同

**批评观点的特别标准:**
| 标准 | 含义 |
|------|------|
| **知识不足** | 作者缺乏某些重要相关事实 |
| **知识错误** | 命题不合事实或自相矛盾 |
| **不合逻辑** | 论点推不出结论，或前后矛盾 |
| **分析与理由不完整** | 没有解决所有问题，或论证不充分 |

> ⚠️ 前三点是表示不同意见的准则，无法提出佐证就必须同意作者说法。你只能因为最后一点暂缓评论。

**同时回答:**
- **作者的时代局限**: 这本书写于什么时期？哪些前提可能已经不成立？
- **作者的立场盲点**: 作者的身份/行业/文化背景让他忽略了什么？
- **未被证明的假设**: 作者把什么当成了不言自明但其实需要论证的东西？

这一步的产出会成为每个总结单元的 **Boundary (B)** 字段来源。

### 步骤 4 — 总结潜力评估 (Applicability)

- **哪些内容值得提炼为独立总结单元？** — 框架/清单/原则/决策程序
- **哪些内容不适合提炼？** — 纯史料/纯故事/纯情感
- **预估总结单元数量**: 给一个粗略区间
- **预估优先级**: 从"最能赋能普通人"的角度排序

---

## 步骤 5 — 书籍分类 + 长书判断（本步骤常被忽略）

**书籍类型判断**（从"书的类型"中选择）:

| 类型 | 特征描述 | 重点 extractor |
|---|---|---|
| 方法论专著 | 有完整体系，大量框架/原则/推理过程 | 全部认真跑 |
| 散文/随笔集 | 碎片化洞见，少体系 | framework + principle 为主；case 轻量 |
| 学术/理论著作 | 术语密集，论证严密 | glossary + framework 为主 |
| 传记/案例集 | 大量真实事件叙述 | case 为主；framework 轻量 |
| 虚构作品（小说/戏剧/诗歌） | 无方法论骨架，思想是文学性的 | **不适用**：仅输出 INDEX.md 四问导读 |
| 其他（画册/诗集/菜谱） | 无文本论证结构 | **不适用**：告知用户不适合方法论提炼 |

**判定为"不适用"时的操作**: 如书籍为虚构或"其他"类型，立即告知用户原因，**不进入阶段 2-4**。如用户坚持处理虚构作品，可只输出 INDEX.md（含 Adler 四问导读），但明确标注"本书为虚构作品，不包含可蒸馏的方法论单元"。

**长书判断**: 文本超过 5 万字时，标记"长书"，在阶段 2 使用分块策略。

## 常见失败模式

1. **跳过批判阶段** — 导致总结把作者的偏见当真理
2. **骨架不是"作者的骨架"，而是自己的想法** — 注意你在写摘要还是在写读后感
3. **术语定义用字典/常识而非作者的特定用法** — 作者用"能力圈"和字典用"能力圈"不是一回事
4. **没有足够证据就评价** — "我觉得"不等于"我证明了"
5. **跳过解释阶段直接提取** — 关键术语不理解，提取就会走偏
6. **不做书籍类型判断** — 导致阶段 2 平均用力，效率低下
7. **阶段 1 批判性评价混入阶段 4 输出** — 两者目的不同，不能混用
