# 步骤3：监管政策分析

## 输入
- 《商业银行授信工作尽职指引》
- 《贷款风险分类指引》
- 《产业结构调整指导目录》（现行版）
- 工信部《重点行业领域产能过剩预警机制》
- 国家统计局《国民经济行业分类》（GB/T 4754-2017）

## 过程
1. 研读监管文件，提取与行业分析相关的核心要求
2. 识别行业准入的监管红线和禁止性行为
3. 分析《产业结构调整指导目录》的鼓励类/限制类/淘汰类分类标准
4. 建立产能过剩预警指标体系（产能利用率 < 60% 且连续2年下降）
5. 设计行业红线核查流程（6条红线 I1-I6）

## 输出
- 行业准入监管要求清单（已整合至 SKILL.md Constraints）
- 6条行业红线定义（I1-I6，已整合至 SKILL.md）
- 产能过剩预警指标体系（已整合至 SKILL.md 步骤6）
- 行业分类标准（GB/T 4754-2017，已引用至 references/industry-analysis-guide.md）

## 关键发现
1. 监管要求行业分析必须覆盖政策环境、产能周期、竞争格局三个核心维度
2. 淘汰类行业和严重产能过剩行业必须暂停授信，这是硬性红线
3. 产能利用率 < 60% 且连续2年下降是严重产能过剩的判定标准
4. 行业分析必须明确界定行业口径（GB/T 4754-2017 四位代码），避免混同分析
5. 数据时效性要求：超过2年的数据必须标注"历史参考"
