# jw-daily-market-judgment

A股每日大势研判

## 简介

面向A股的盘后大势研判技能：基于数据、事件、情绪、资金和板块轮动，输出结构清晰、结论先行的市场走势理解。适用于用户询问'今天大盘怎么看''市场处于什么阶段''主线是什么''情绪周期到哪了''盘后复盘''北向资金怎么看''大盘今天怎么样''现在是什么行情''市场强不强''今天涨了还是跌了''主线是谁''情绪到哪了'等场景。
