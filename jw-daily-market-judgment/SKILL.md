# A股每日大势研判

## 0. 触发条件

当用户出现以下任一需求时，使用这个 skill：
- 让你研判今天/明天的 A股大盘
- 让你判断当前市场处于什么阶段
- 让你分析主线板块、资金流向、情绪周期或轮动结构
- 让你输出盘后复盘、盘前判断、次日观察点
- 让你判断该加仓、减仓、观望还是等待确认

## 1. 目标

输出要回答四个问题：
1. 当前市场处于什么状态？
2. 今天是谁在主导市场？
3. 主线板块和轮动方向是什么？
4. 明天应该怎么做，什么情况下判断失效？

## 2. 适用范围

只适用于 A股市场。不适用于港股、美股、纯个股价值分析。

## 3. 依赖技能

- **jw-investment-data**（路径：`/root/.hermes/skills/jw-investment-data/SKILL.md`）：行情与结构数据底座（指数、K线、涨跌家数、成交额、融资融券、北向资金）
- **jw-news**（路径：`/root/.hermes/skills/jw-news/SKILL.md`）：事件信息层（政策、新闻、财经日历）
- **financial-data-mcp**（路径：`/root/.hermes/skills/data-science/financial-data-mcp/SKILL.md`）：实时补充与校验层（金十快讯、实时行情）

**降级策略**：如果某个依赖技能不可用或数据获取失败，用 `web-access` 搜索东方财富、同花顺等财经网站补充。如果数据仍不足，明确标注"数据不足，以下判断为低置信度"。

## 4. 研判流程（8步）

执行顺序固定，每一步的结论喂给下一步。步骤之间有依赖关系，不要跳步。

**场景区分**：
- **盘中实时研判**（步骤1-6即可）：快速判断，输出精简版结论
- **盘后深度复盘**（完整步骤1-8）：完整分析，输出详细报告

### 步骤1：数据采集

**数据来源**：
- 如果用户没有显式提供数据，调用 jw-investment-data 和 jw-news 采集
- 如果数据不足，明确说明"数据不足，以下判断为低置信度"

**采集内容**：
- 今日主要指数表现（上证、深证、创业板、科创50、沪深300）
- 成交额与较前日变化（**相对值比绝对值更重要**）
- 近10个交易日的成交额序列
- 涨跌家数、涨停家数、跌停家数、炸板率
- 涨跌家数比例（如 1:2.5）
- 连板梯队分层（首板、2板、3板、高度板）
- 板块涨幅排行
- **融资融券余额变化**（杠杆资金动向）
- **北向资金净流入/流出**（外资动向）
- 重要新闻、政策、事件

### 步骤2：事件驱动判断

**事件分级**：

| 级别 | 举例 | 影响力 |
|------|------|--------|
| 重大 | 央行降息/降准、国务院级别政策、战争爆发、GDP大幅超/不及预期 | 可能改变市场趋势 |
| 中等 | 部委级政策、行业规划、重要公司财报、PMI/CPI偏离预期 | 影响相关板块 |
| 轻微 | 地方政策、个别公司事件、媒体吹风 | 有限影响，可能被忽略 |

**事件影响判断**：
```
如果 事件级别 >= 重大 且 市场未充分反应：
  → 可能触发趋势改变，需要调整整体仓位

如果 事件级别 = 中等 且 市场未充分反应：
  → 影响相关板块，调整板块配置

如果 事件 = 利好落地 且 市场已充分反应：
  → "买预期卖事实"，警惕冲高回落

如果 事件 = 利空落地 且 市场已充分反应：
  → "利空出尽"，可能触底反弹
```

**事件脱敏效应**：
- 同一类型事件多次发生后，市场反应会钝化
- 判断标准：如果该事件在过去30天内发生过3次以上，视为已脱敏，影响力度降一级

### 步骤3：市场状态、情绪与风格判断（核心）

将市场状态、情绪周期、风格判断合并为一步，因为三者高度耦合——市场状态是情绪的外在表现，风格是情绪的传导机制。

**第一层：市场状态判断**

| 状态 | 核心特征 | 典型情绪阶段 |
|------|----------|-------------|
| 趋势市 | 指数、广度、量能三者共振，主线明确 | 高潮→延续 |
| 震荡市 | 指数区间波动，板块轮动频繁 | 修复/混沌 |
| 修复市 | 前期退潮后情绪回暖，但未确认趋势 | 冰点→修复 |
| 退潮市 | 涨停减少、跌停增加、资金流出 | 退潮→冰点 |
| 混沌市 | 没有明确主线，轮动快，强弱切换 | 混沌 |

**决策树（使用相对值，不依赖绝对阈值）**：
```
如果 涨停家数 > 20日均值的1.5倍 且 跌停 < 20日均值的0.5倍
   且 成交额 > 20日均值 且 较前日放量10%以上
   且 首板晋级率 > 30%：
  → 趋势市

如果 涨停家数在20日均值附近（0.7~1.3倍）
   且 跌停在20日均值附近
   且 成交额在20日均值附近
   且 涨跌家数比例在 1:1 ~ 1:2 之间：
  → 震荡市或修复市

如果 涨停家数 < 20日均值的0.5倍 且 跌停 > 20日均值的2倍
   且 成交额 < 20日均值的0.8倍
   且 涨跌家数比例 > 1:3：
  → 退潮市或混沌市
```

**模糊区间处理**：如果多个信号指向不同状态（如涨停多但跌停也在增加），不要强行归类，而是写明"当前处于 [状态A] 和 [状态B] 的模糊地带，信号冲突点在于 xxx"，仓位取两个状态的保守值。

**优先级规则**：
- 如果多个信号冲突，以"涨停家数/跌停家数"比例为主
- 成交额的相对变化比绝对值更重要

**第二层：情绪周期细分**

| 阶段 | 关键信号 | 仓位建议 |
|------|----------|----------|
| 冰点 | 跌停/亏钱效应显著，首板少，高度板断裂 | 0-3成 |
| 修复 | 情绪开始回升，首板增多，但连板晋级率低 | 3-5成 |
| 高潮 | 涨停多、连板高、情绪亢奋、首板晋级率高 | 5-7成，警惕退潮 |
| 退潮 | 高度板断裂、炸板率高、亏钱效应扩散 | 0-3成 |
| 混沌 | 无一致方向，强弱切换快，轮动无序 | 0-3成 |

**情绪尾部特征**（退潮前兆，出现2个以上就要警惕）：
- 地天板/天天地频繁出现
- 龙头复盘后补涨股集体兑现
- 首板晋级二板概率持续 < 10%
- 高度板断裂（三进四没了）
- 周一情绪本应最好但仍走弱

**情绪尾部操作**：
- 仓位降到两三成
- 只做龙头，"要死也死在龙头上"
- 杂毛股不要碰

**第三层：风格判断**

判断当前是量化主导还是游资主导：

| 维度 | 量化主导 | 游资主导 |
|------|----------|----------|
| 开盘表现 | 联动下跌，不给反弹 | 有犹豫、有试探、有从容卖点 |
| 首板晋级率 | < 10% | > 30% |
| 高度板 | 断裂（三进四没了） | 可突破八板 |
| 卖点 | 确认即走，不能等反弹 | 可等分时高点 |

**风格优先级**：风格判断会覆盖前两层的部分建议：
- 量化主导：即使市场状态是"趋势市"，短线仓位也不超过5成，不追高不抄底
- 游资主导：连板活跃，龙头战法有效

**最终仓位建议**：取三层的**交集**。
- 三层一致（如趋势市+高潮+游资主导）→ 激进，按上限执行
- 两层一致、一层冲突 → 取中间值
- 三层冲突或信号模糊 → 取保守值，仓位不超过3成

### 步骤4：资金面判断

**判断资金流向**：

| 资金状态 | 特征 | 操作建议 |
|----------|------|----------|
| 持续流入 | 成交额放大、热点集中、主线明确 | 5-7成，跟随主线 |
| 脉冲式流入 | 单日放量但不持续 | 3-5成，观察持续性 |
| 选择性流入 | 只有部分板块涨，其他不动 | 3-5成，只做强势板块 |
| 退潮式流出 | 成交额萎缩、热点分散 | 0-3成，减仓防守 |
| 存量博弈 | 成交额稳定、板块轮动快 | 3-5成，高抛低吸 |

**关键指标**（使用相对值）：
- 成交额较20日均值的变化比绝对值更重要
- 持续缩量后放量（较前日+20%以上）：可能是抄底信号
- 持续放量后缩量（较前日-20%以上）：可能是见顶信号

**外资动向判断标准**：
- 北向资金单日净流入 > 50亿 → 明显加仓，市场有支撑
- 北向资金单日净流出 > 50亿 → 明显撤退，风险较大
- 连续3天同方向流动 → 趋势信号，比单日更有意义

**融资融券**：
- 融资余额持续增加 → 杠杆资金进场，偏多
- 融资余额持续减少 → 杠杆资金撤退，偏空
- 融券余额突增 → 做空力量增强，警惕

### 步骤5：板块轮动与技术面判断

**板块生命周期**：

| 阶段 | 特征 | 操作建议 |
|------|------|----------|
| 预期酝酿 | 有催化但还没涨 | 观察，不追 |
| 启动 | 首板出现、趋势突破 | 试错建仓 |
| 加速 | 连板增多、资金涌入 | 跟随，不追情绪尾部 |
| 高潮 | 涨停最多、情绪亢奋 | 警惕退潮 |
| 退潮 | 高度板断裂、炸板增多 | 减仓，观察反抽质量 |
| 反抽 | 退潮后短暂反弹 | 只做龙头，快进快出 |
| 结束 | 没有新催化 | 彻底离开 |

**轮动速度判断**：
- 轮动过快（热点日内切换2次以上）→ 混沌市特征，不宜追热点
- 轮动有序（主线持续3天以上）→ 趋势市特征，可以跟随主线

**板块锚点思维**：
- 美股核心标的（如英伟达、谷歌）对A股相关板块有定价指引
- 谷歌走势 → AI应用方向
- 英伟达走势 → 算力方向

**技术面判断**（只看结构，不看单根K线）：

均线系统：
- 5日线：短线生命线，跌破减仓，站稳持有
- 10日线：短期趋势线，是短线回踩的常见支撑
- 20日线：中期趋势线，是波段操作的锚点
- 60日线：牛熊分界线，跌破意味着中期趋势转弱

结构定义：
- **上升结构**：高点抬高、低点抬高、站在5/10/20日线之上
- **下降结构**：高点降低、低点降低、被5/10/20日线压制
- **箱体结构**：在明确的上下沿之间震荡，均线缠绕

量价关系：
- 放量突破 → 有效突破概率高
- 缩量突破 → 假突破概率高
- 放量下跌 → 恐慌盘释放，可能是阶段性底部
- 缩量下跌 → 阴跌不止，底部未到

**九转序列使用规则**：
- 前提：先确定大趋势方向
- 主升正向波：高九容易失效（不必减仓），低九容易失效（不必抄底）
- 主升逆向波：低九可作为反弹买入信号，高九可作为卖出信号
- 判断口诀：正向波的九转容易失效，逆向波的九转才有效

**MACD金叉**：
- 60分钟级别出现金叉+红角线 → 短期反弹信号
- 反弹周期约12个交易日

### 步骤6：仓位管理与综合研判

**仓位决策树**（综合步骤3-5的所有信号）：

```
信号一致（全部偏多/全部偏空）→ 可以激进，按步骤3的仓位建议执行
信号冲突（部分偏多部分偏空）→ 取保守值，降低1-2成仓位
数据不足或判断模糊 → 观望为主，仓位不超过3成
```

**仓位调整的额外规则**：
- 量化主导时：即使信号偏多，短线仓位也不超过5成
- 退潮市+冰点：无论其他信号如何，仓位不超过3成
- 风格切换确认当天：仓位降低1-2成，等新风格稳定后再加

**一句话结论的写法**：
- 优先级：市场状态 > 主线方向 > 策略倾向
- 格式：`市场处于 [状态] 阶段，主线是 [板块]，当前策略偏 [激进/稳健/防守]，重点观察 [关键变量] 是否延续。`
- 不要用"关注""留意""等待"等空话，必须给出明确的方向判断

**分层输出**：

核心层（盘中快速版，3秒看完）：
```
【结论】市场处于 xxx 阶段，策略偏 xxx，仓位 xxx 成
【动作】xxx
```

完整层（盘后复盘版）：
```
【一句话结论】
市场处于 xxx 阶段，主线是 xxx，当前策略偏 xxx，重点观察 xxx 是否延续。

【市场状态】xxx（趋势/震荡/修复/退潮/混沌）
【情绪周期】xxx（冰点/修复/高潮/退潮/混沌）
【风格判断】xxx 主导（量化/游资）
【资金面】xxx（持续流入/脉冲/选择性/退潮/存量博弈）
【主线板块】xxx
【技术面】xxx（结构 + 均线 + 量价）

【仓位建议】xxx 成

【失效条件】
```

## 5. 盘前模板

**使用时机**：开盘前或集合竞价后，需要提前判断当日策略时使用。

**执行顺序**：先完成步骤1-3（数据采集→事件判断→市场状态+情绪+风格），再根据结果预写剧本。

**盘前剧本预写（强制）**：

**情况一：高预期**
- 触发条件：如"昨天大涨+盘前利好，今天应该高开1%以上"
- 应对策略：如"继续持有，观察量能是否配合"
- 止盈条件：如"如果冲高回落跌破开盘价，减仓30%"

**情况二：符合预期**
- 触发条件：如"昨天大涨+盘前利好，今天平开或小幅高开"
- 应对策略：如"观察盘中表现，如果10:30前不能创新高，考虑减仓"
- 止盈条件：如"跌破昨日收盘价，减仓50%"

**情况三：低预期**
- 触发条件：如"昨天大涨+盘前利好，今天低开"
- 应对策略：如"开盘即确认方向不对，立即减仓50%以上"
- 止损条件：如"跌破关键支撑位，清仓"

**开盘验证**：
- 开盘5分钟内观察：是否符合预期？
- 开盘30分钟内确认：高预期/符合预期/低预期？
- 根据剧本执行，不要犹豫

## 6. 盘后模板

**使用时机**：收盘后复盘、总结市场状态、为次日做准备时使用。

**执行顺序**：完成步骤1-6的完整流程后，按以下顺序输出：

1. 今日一句话结论
2. 今日市场环境（状态+情绪+风格）
3. 今日资金面变化（成交额+北向+融资融券）
4. 今日主线与轮动
5. 今日仓位与动作建议
6. 今日失效条件
7. 次日观察点

### 失效与降级规则

如果出现以下情况，主动降级结论：
- 关键数据缺失
- 盘面与事件完全背离
- 市场风格切换过快，无法确认主线
- 指数与个股广度严重背离
- 信号冲突且无法判断优先级

降级时明确写：
- 目前只能给出低置信度判断
- 暂时不适合下重仓结论
- 需要等待下一交易日验证

### 置信度规则

| 置信度 | 条件 |
|--------|------|
| 高 | 指数结构清晰、广度和量能配合、主线明确、资金持续、事件与盘面一致 |
| 中 | 满足大部分条件，但有一到两个关键变量未确认 |
| 低 | 数据缺失、盘面冲突明显、主线不清、情绪极端但缺乏延续 |

### 避免错误

1. 只看指数，不看广度
2. 只看单日涨跌，不看结构
3. 把脉冲行情当趋势
4. 把情绪高潮当主升起点
5. 把事件利好等同于盘面利好
6. 把局部抱团误判为全面修复
7. 不写失效条件
8. 不给动作建议
9. 用空话代替判断
10. 在量化主导环境下还用游资思维做短线
11. 不区分主升正向波和逆向波就乱用九转序列
12. 忽略事件脱敏效应，对重复事件过度反应
13. 不写风格判断，不知道当前是量化还是游资主导
14. 信号冲突时强行归类，不写模糊地带

## 8. 最终原则

- 先识别市场状态
- 再识别主导力量
- 再识别资金方向
- 再给动作建议
- 最后写清失效条件

宁可保守，不要硬猜。
宁可少说，不要空说。
宁可低置信度，也不要伪确定。
信号冲突时说清楚冲突在哪里，不要装作一切清晰。
