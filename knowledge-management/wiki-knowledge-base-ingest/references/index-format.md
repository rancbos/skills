# index.md Wikilink 格式规范

## 标准条目格式

```markdown
|[[entity_id]] 书名 — 一句话简介
```

示例：
```markdown
|[[buffett-zhi-dao]] 巴菲特之道 — 购买企业12准则、9个普通股投资案例
```

## 分类排序

每个分类内按以下顺序排列：
1. 经典核心典籍（儒家十三经等）
2. 重要派生著作
3. 现代研究/通俗读物
4. 参考性条目

## 分类头部格式

```markdown
### 价值投资（invest）— 67本
```

格式：`### {分类中文名}（{分类ID}）— {本数}本`

## 分类分隔

分类之间用空行隔开。每个分类内部条目每行一个，空行分隔。

## 别名格式

两个 entity_id 指向同一实体时用别名：

```markdown
|[[zuozhuan]] 左传 — 春秋左氏传，叙事详备
|[[liezhuan]] 烈女传 → [[lienvzhuan]]（别名）
```

## 注意事项

- Wikilink 使用 `[[entity_id]]` 格式，entity_id 不含空格，全小写
- 标题中的书名号 `《》` 应去掉，只保留书名正文
- 作者信息放在"一句话简介"之后，用破折号分隔：`| 书名 — 简介 — {作者}`（可选）
- 定期从 index 统计总条目数，更新第一行的 `总计：{N}本典籍`
