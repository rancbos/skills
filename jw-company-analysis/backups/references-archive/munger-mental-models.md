# Step 5：逆向检查与误判防护（安全阀）

### 5.1 芒格误判心理学检查（10项核心）

| # | 误判倾向 | 检查问题 |
|---|---------|---------|
| 1 | 激励偏见 | 管理层激励是否对齐股东利益？ |
| 2 | 确认偏见 | 我是否只看支持自己观点的信息？ |
| 3 | 锚定效应 | 我是否被某个数字锚定了？ |
| 4 | 社会认同 | 我是否因为别人都买才想买？ |
| 5 | 对比效应 | 我是否因为比同行便宜就觉得便宜？ |
| 6 | 压力效应 | 我是否在时间压力下做决定？ |
| 7 | 易得性偏见 | 我是否只记得最近的信息而忽略长期？ |
| 8 | 权威偏见 | 我是否因为大佬推荐就想买？ |
| 9 | 损失厌恶 | 我是否因为不想割肉而持有烂公司？ |
| 10 | 过度自信 | 我是否高估了自己的判断能力？ |

**25种误判倾向完整体系**（芒格）：核心10项已列于上表，完整25项🧠详见munger-mental-models.md 第十一讲。

**叠加扣分机制**（含严重程度权重）：

**Step 1：评估每项偏见的严重程度**：

| 严重程度 | 定义 | 权重系数 | 典型表现 |
|---------|------|---------|---------|
| 轻微 | 存在偏见倾向但不影响核心判断 | ×0.5 | "可能过度关注NBV高增长" |
| 中等 | 偏见影响了部分评分但未改变结论 | ×1.0 | "被PEV低值吸引，未充分考虑利率风险" |
| 严重 | 偏见直接导致判断错误或忽视致命风险 | ×2.0 | "无视财务造假迹象""忽略行业颠覆性变化" |

**Step 2：按加权后总偏见数查表扣分**：

| 轻微数 | 中等数 | 严重数 | 加权总数 | 扣分 |
|--------|--------|--------|---------|------|
| 3 | 0 | 0 | 1.5→2 | -12分 |
| 5 | 0 | 0 | 2.5→3 | -21分 |
| 0 | 3 | 0 | 3 | -21分 |
| 6 | 0 | 0 | 3→3 | -21分 |
| 3 | 2 | 0 | 3.5→4 | -32分 |
| 0 | 0 | 3 | 6→6 | -50分 |
| 5 | 1 | 0 | 3.5→4 | -32分 |
| 4 | 0 | 1 | 4→4 | -32分 |

**计算公式**：`加权总数 = Σ(偏见数 × 权重系数)`，四舍五入取整后查标准扣分表（1=-5, 2=-12, 3=-21, 4=-32, 5=-45, 6+=-50）。

**核心改进**：6项轻微偏见（6×0.5=3）→ 扣21分（而非原来的-50分上限）。只有存在严重偏见时才会接近-50上限。这确保"认真做逆向检查但发现的都是轻微偏见"不会自动触发人工复核。

**⚠️ Pitfall：偏见严重程度评估纪律**：
- 不得自行降低严重程度以规避扣分——这是偏见的偏见
- 若不确定严重程度，取更高一级
- "轻微"偏见≥6项时，需反思分析过程是否存在系统性盲区（即使扣分仅-21）

### 5.2 逆向投资检查

> **本节包含两个独立子得分**，需分别评估后汇总。

**子得分A：逆向检查（+10或-10分）**
- **通过**（+10分）：最大风险可承受、最坏情况可存活、不存在致命隐患
- **不通过**（-10分）：存在致命风险（如财务造假迹象、行业颠覆性变化、管理层诚信问题）

**子得分B：市场情绪（+10或-10分）**
- **市场恐惧**（+10分）：大众恐慌、估值被压制、逆向机会
- **市场贪婪**（-10分）：全民热议、估值泡沫化、需谨慎
- **中性**（0分）：无明显情绪偏向

**投机vs投资定义**（格雷厄姆）：基于全面分析确保本金安全且收益合理=投资，否则=投机。

**反过来想**：不买可能因为什么？卖了可能因为什么？

**最大风险**：最大风险是什么？概率多大？损失多少？

**最坏情况**：损失多少？能承受吗？公司还能活下来吗？

**鸡尾酒会理论**（彼得林奇）：四阶段判断市场情绪——①无人谈股票=底部 ②少数人谈=复苏 ③人人谈=过热 ④出租车司机荐股=见顶。

**市场周期钟摆**（马克斯）：市场情绪在贪婪-恐惧间摆动，钟摆很少停在中点，大部分时间在两极。

**第二层思维**（马克斯）：第一层"公司好→买"，第二层"公司好→共识→已定价→不买"。

**冯柳"降维"操作策略**：更多研究是为了更少的决策——广泛涉猎升维→聚焦3个关键变量→极度克制决策次数。

**风险态度周期**（马克斯）：市场风险态度在过度规避和过度容忍间摆动，过度容忍时风险最高。

**信贷周期**（马克斯）：信贷周期是经济周期的放大器——繁荣期信贷宽松→泡沫→收缩期信贷收紧→萧条。

**泡沫九特征**（特维德）：新事物+融资容易+新模型+价格飞涨+大众参与+批判者被嘲笑+信贷扩张+新范式+崩盘。

**周期定位信号**（特维德）：用利率/信贷/库存/情绪四维度定位当前周期位置。

**市场先生寓言**（格雷厄姆/巴菲特）：三层应用——①不被市场情绪左右 ②利用市场愚蠢 ③股价偏离≠基本面变化。

> **宏观周期背景**：详见 Step 0.6 宏观环境评估。

**评分标准**：误判检查按 5.1 叠加扣分机制（含严重程度权重，-5/-12/-21/-32/-45/-50）；逆向检查（子得分A）+10或-10；市场情绪（子得分B）+10或-10。

**得分范围限制**：Step 5 总分 = 误判检查扣分 + 逆向检查得分 + 市场情绪得分 + ESG扣分，硬限在 -50到+20 范围内。

**Step 5 得分组合公式**：
```
Step 5 总分 = min(20, max(-50, 误判扣分 + 逆向检查(±10) + 市场情绪(±10/0) + ESG扣分(0/-5/-10)))
```
总分硬限 -50到+20。总分 < -20 时触发人工复核，报告中必须单独列出风险警示。

### 5.3 ESG风险评估

ESG风险可能对投资产生重大影响，需在逆向检查中重点关注：
- **环境（E）**：碳排放政策风险、环保合规风险、气候变化物理风险
- **社会（S）**：劳工权益、产品安全、数据隐私
- **治理（G）**：董事会独立性、关联交易（>20%警惕）、大股东质押率（>50%风险高，>80%排除）、审计意见（非标直接排除）、重大治理缺陷（如实际控制人被调查）直接排除

**ESG 评分影响**：低风险+0分 / 中风险-5分 / 高风险-10分。**ESG排除条件**（触发任一直接排除，不走扣分路径）：审计意见非标、大股东质押率>80%、实际控制人被调查/立案、重大环保/安全事故。ESG扣分仅在不触发排除条件时适用，与误判检查叠加，总扣分不超过50。

### 5.4 风险分类框架

> **与5.1/5.2/5.3的分工**：5.1检查投资者心理偏见，5.2检查逆向投资逻辑+市场情绪，5.3检查ESG合规风险。本节聚焦**企业经营层面的四类风险**，与前三节互补不重复。评估结果写入step5_contrarian.json的`risk_classification`字段。

**①行业风险**（→ 引用 Step 3.0.5 结论，不重复评估）：景气度下行 / 竞争加剧 / 技术替代 / 成本上升。

**②公司风险**：大客户流失（前5大客户集中度>50%警惕）/ 核心产品失速 / 新业务不及预期 / 产能扩张过快（产能利用率<70%）/ 海外业务风险。

**③财务风险**（→ 引用 Step 2 结论，不重复评估）：负债过高 / 回款变差 / 商誉减值 / 资本开支过大。

**④管理风险**（→ 引用 Step 1.3 结论，不重复评估）：战略失误 / 激进并购 / 治理问题 / 信息披露风险。

**风险等级**：低风险（各项均绿）→ 中风险（个别黄灯）→ 高风险（多项黄灯或单红灯）→ 排除（触发硬性排除条件）。
