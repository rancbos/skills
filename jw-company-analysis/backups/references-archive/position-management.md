# Step 7：综合评分与投资结论

> **输入**：Step 1-6 全部评分 + Step 4 估值区间 → **输出**：加权总分 + 投资结论 + 建仓建议

**前景预测三步法**（张新民）：①经营活动看前景 ②投资活动看前景 ③融资/重组/并购看前景。

**辩论记录**（Bull→Bear→Synthesis）：在Step 7综合评分后执行。

**三轮流程**：
```
Bull 发言（看多，必须引用数据）→ Bear 发言（看空，必须引用Bull首句并逐一反驳）→ Synthesis（裁判）
```

**Bull 分析要点**：增长潜力 / 竞争优势（护城河）/ 正向指标（财务健康/行业顺风/近期催化）/ 预判熊方论点并预先反驳

**Bear 分析要点**：风险与逆风 / 竞争弱点 / 负面指标 / **逐条反驳Bull**（必须引用Bull的具体论点）

**Synthesis 固定输出5段格式**：

```
**Rating:** <⭐强烈推荐 / 🟢推荐 / 🟡观望 / 🔴不推荐 / ❌远离>

**Thesis (2-4 sentences):** 支持评级的最强单一证据是什么

**What would change the view:**
- 升级条件：<具体可观测阈值/事件>
- 降级条件：<具体可观测阈值/事件>

**Risks acknowledged:** 评级无法完全消化的最强1-2个风险点

**Holding period and re-check trigger:** 目标持有期 + 触发下次重新评估的精确事件/指标
```

### 7.1 加权总分计算
（公式见文件开头"统一权重表"章节，此处不重复。）

**⚠️ 行业校准调整规则**（v3.56新增，强制执行）：

> **核心原则**：综合得分必须反映行业特性，但调整必须有规则依据，**禁止手动凭感觉调分**。

**适用场景**：当 Step 0.4.1 判定的 `industry_type` 为金融/周期等行业时，以下规则自动激活：

| 行业类型 | 校准规则 | 调整幅度 | 依据 |
|---------|---------|---------|------|
| **金融-银行** | Step 2 资产负债率标准上调至70%（行业常态），有息负债率标准上调至50% | Step 2 得分修正（不超过+5分） | 银行高杠杆是商业模式而非风险 |
| **金融-保险** | Step 2 使用偿付能力充足率替代资产负债率标准，Step 4 使用PEV替代PE | Step 2+4 得分修正 | 保险负债驱动，传统指标失真 |
| **金融-券商** | Step 2 使用净资本/净资产替代流动比率，Step 4 使用PB+量能 | Step 2+4 得分修正 | 券商轻资产+强周期 |
| **周期-资源** | Step 1.4 成长性用5-10年周期均值替代当期增速，Step 4 使用正常化PE | Step 1.4+4 得分修正 | 周期利润波动大，当期不具代表性 |
| **科技-成长** | Step 1.2 护城河"宽度"放宽至5年（科技行业变化快），Step 2 ROE标准放宽至>8% | Step 1.2+2 得分修正 | 科技研发投入高，ROE天然偏低 |
| **金融-地产** | Step 2 资产负债率标准上调至70%，有息负债率标准上调至50% | Step 2 得分修正 | 地产高杠杆是行业常态 |
| **消费-品牌** | Step 1.2 护城河重点关注品牌+渠道（消费者垄断型），Step 4 优先DCF | 不额外调整 | 消费品牌是标准评估对象 |

**执行规则**：
1. 校准调整**仅在对应Step的评分计算后**生效，修改该Step的最终得分
2. 调整幅度**有上限**：每个Step最多修正+5分（向上修正），向下修正无上限
3. 修正后必须在报告中标注：`[行业校准：XX行业，原始XX分→校准后XX分，依据：YY]`
4. 若 `industry_type` 无法匹配上表 → 不做校准，按默认标准评分

**禁止的调分行为**：
- ❌ "这家公司PEV很低所以综合得分应该更高"（必须走行业校准表）
- ❌ "保险行业特殊，手动加5分"（必须有上表规则依据）
- ❌ "我觉得市场低估了，调整安全边际"（安全边际由公式计算，不手动调整）

### 7.2 投资结论矩阵

> **分数校准说明**：综合得分公式见文件开头"统一权重表"章节。总范围-5到92分，实际有效范围约40-92分。
> 评分参考：四维均90+ → ~85-92（优秀）；四维均80 → ~72（好标的）；四维均60 → ~52（一般）。

| 综合得分 | 结论 | 持有建议 | 建仓建议 |
|---------|------|---------|---------|
| ≥ 80 | ⭐ 优秀标的 | 可长期重仓持有 | 当前价位可建仓 |
| 65-79 | 🟢 好标的 | 标准仓位，定期审视 | 等回调至安全边际内建仓 |
| 50-64 | 🟡 观望 | 不建议买入，继续跟踪 | 暂不建仓 |
| < 50 | 🔴 不推荐 | 不符合价值投资标准 | 不建仓 |

### 7.3 风险警示

> **与排除检查汇总表的分工**：汇总表定义"排除/降级"的硬性触发条件（Step 0-5的执行层面）；本节定义"报告中需要特别关注"的风险项（Step 7的投资决策层面）。两处互补，不重复。

以下任一条件触发，需特别关注并谨慎决策：

- ⚠️ **护城河评分 < 10**（没有持久竞争优势）→ 建议谨慎
- ⚠️ **管理层诚信评分 ≤ 5**（Step 1.3中"诚信与透明度"维度得分≤1分）→ 建议谨慎
- ⚠️ **连续3年经营现金流为负**（造血能力丧失）→ 需要特别关注
- ⚠️ **财务造假/重大违规历史**（诚信红线）→ 需要特别关注
- ⚠️ **有息负债率 > 80% 且无合理解释**（行业校准：银行/保险不适用；房地产60-80%为常态）→ 需要特别关注
- ⚠️ **商誉/净资产 > 50%**（减值风险过高）→ 需要特别关注

### 7.4 催化剂分析

> 催化剂 = 市场重新定价公司的触发事件。好公司+好价格+催化剂 = 最佳买入时机。

**7类催化剂识别清单**：

| # | 催化剂类型 | 触发信号 | 预期影响 |
|---|-----------|---------|---------|
| 1 | 业绩超预期 | 季度/年度业绩超出市场一致预期10%+ | 股价跳涨+估值修复 |
| 2 | 新产品发布 | 重磅产品上市/技术突破 | 打开新增长曲线 |
| 3 | 关键客户突破 | 进入大客户供应链/签订重大合同 | 收入确定性提升 |
| 4 | 行业景气回升 | 行业PMI转正/库存去化完成/价格企稳 | 周期股反转 |
| 5 | 成本拐点出现 | 原材料价格回落/规模效应显现 | 毛利率改善 |
| 6 | 估值切换 | 市场从"当期PE"切换到"明年PE" | 高成长股估值提升 |
| 7 | 资本动作 | 大额回购/增持/分红提升/股权激励 | 市场信心修复 |

**催化剂评估**：是否存在催化剂 / 催化剂何时触发 / 催化剂强度（强/中/弱）/ 是否已被市场定价。

**催化剂+估值联动**：存在强催化剂 → 可在估值区间内偏积极建仓；无催化剂 → 按估值建议耐心等待。

### 7.5 建仓建议

**建仓终检清单**（12项，全部通过才可建仓）：
- [ ] ① 能力圈：我能理解这家企业？（5分钟测试通过）
- [ ] ② 护城河：有持久竞争优势？（四问检验法通过）
- [ ] ③ 管理层：诚信能干？（荒岛两问通过+资本配置理性）
- [ ] ④ 利润三大前提通过？（为真/可持续/不需大量资本）
- [ ] ⑤ 安全边际充足？（A股>30%，一般企业>40%）
- [ ] ⑥ 逆向检查通过？（无重大偏见+无财务爆雷信号）
- [ ] ⑦ 机会成本胜出？（比现有最佳持仓更好）
- [ ] ⑧ 风险评估：永久性资本损失风险可接受？
- [ ] ⑨ 能力圈边界：不在变化迅速的行业？不投商品型企业？
- [ ] ⑩ 杠杆检查：不用借钱买股票？不做空？
- [ ] ⑪ 管理层激励：激励机制与股东利益一致？
- [ ] ⑫ 核心持股标准：可以长期持有？（非短期交易）

**建仓区间**：下限=内在价值×(1-安全边际要求)，上限=内在价值×(1-最低安全边际)

**建仓策略**：首次30% → 下跌10%加至50% → 再跌10%加至70% → 再跌10%满仓（仅综合得分≥80且护城河≥20的标的）

**技术面调节规则**（与 Step 6 联动）：
- 🟢🟢 极佳 / 🟢 良好 → 建仓区间内偏积极，可上调首次建仓比例至40%
- ⚪ 中性 → 按上述标准策略执行
- 🟡 警示 → 建仓区间内偏保守，下调首次建仓比例至20%
- 🔴🔴 极差 / 🔴 不佳 → 暂停建仓，等待技术面改善

> **宏观资产轮动背景**：详见 Step 0.6 宏观环境评估。

**集中投资优于分散**（巴菲特）：优选3-5只深度研究的股票，而非50只浅层分散。

### 7.6 卖出与止损信号

> **卖出决策比买入更重要**。买入错了可以等，卖出错了不可逆。本节建立系统化的卖出决策框架，与 7.5 建仓建议形成完整闭环。

**卖出信号分三类**（按紧急程度排序）：

**🔴 紧急卖出（不等反弹，立即执行）**：
- **财务造假/重大违规**：审计意见变为非标 / 证监会立案调查 / 核心财务数据大幅修正
- **管理层诚信崩塌**：实控人被刑拘 / 重大关联交易隐瞒 / 致股东信被证实造假
- **护城河根本性破坏**：核心技术被替代（如诺基亚案例）/ 特许经营权丧失 / 行业被颠覆
- **利润三大前提全面失效**：经营现金流转负+毛利率骤降+应收暴增（三项同时出现）

**🟡 警示卖出（观察1-2个月，确认趋势后执行）**：
- **基本面趋势性恶化**（连续2个季度）：
  - 营收增速转负（非季节性/非周期性）
  - 毛利率下降>3个百分点（非原材料短期波动）
  - ROE下降>5个百分点（从20%降到15%以下）
  - 经营现金流/净利润<0.7（连续2季）
- **护城河被侵蚀**：
  - 市场份额连续下降（>2个百分点/年）
  - 客户流失率上升（>10%/年）
  - 提价能力丧失（被迫降价促销）
- **管理层信号**：
  - 核心高管集中离职（3人以上/半年内）
  - 大股东持续减持（减持>5%总股本）
  - 频繁跨界并购（偏离主业）
- **估值极端高估**：
  - PE超过历史95%分位且PEG>3
  - 所有者收益收益率<1%（极度高估）
  - 股价超过DCF内在价值上限的150%

**⚪ 持有观察（未触发卖出，但需密切关注）**：
- **成长股PEG监控**（斯莱特）：PEG从<1升至1.5-2区间 → 持有但不加仓；PEG>2 → 准备卖出；配合相对强度转负确认
- **周期股周期位置**：正常化PE超过周期参考区间上限 → 考虑减仓
- **行业政策风险**：监管政策转向但尚未落地 → 密切跟踪

**止损条件**（硬性纪律，不讨论）：
- 价格跌破建仓下限20% → 止损离场，重新评估
- 发现更好机会且资金不足 → 卖出换仓（机会成本止损）
- 买入逻辑被证伪（如看好新品但新品失败）→ 止损

**⚠️ 卖出前的反验证**（防止误卖）：
1. **短期噪音 vs 长期趋势**：当前恶化是短期波动还是长期趋势？（参考2-3年数据判断）
2. **市场情绪 vs 基本面**：股价下跌是因为市场恐慌还是基本面恶化？（参考Step 5.2B市场情绪）
3. **机会成本**：卖出后有更好的标的吗？（无更好选择则持有观察）
4. **税收影响**：持有不足1年卖出需缴税，是否影响净收益？

**股债平衡策略**（格雷厄姆）：标准50:50，范围25%-75%动态调整，股市高估时增加债券比例。
