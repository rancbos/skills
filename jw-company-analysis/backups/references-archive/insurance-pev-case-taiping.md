## PEV估值实战

### 核心计算
```
PEV(2025) = 股价31.05 / 每股EV 63.76 = 0.49倍
PEV(2026E) = 31.05 / 70.30 = 0.44倍（华源证券预测）
PEV(2027E) = 31.05 / 77.18 = 0.40倍（华源证券预测）
```

### PEV参考区间应用
| PEV区间 | 含义 | 中国太保位置 |
|---------|------|-------------|
| <0.5 | 极度低估 | ← 当前0.49（刚好触及） |
| 0.5-0.8 | 低估 | |
| 0.8-1.2 | 合理 | |
| >1.5 | 高估 | |

### 与PB/PE交叉验证
- PE-TTM ~5.5倍（金融行业5-15倍区间，偏低）
- PB 0.93倍（破净，ROE 18%应支撑更高PB）
- 三方法一致指向低估

### 安全边际计算
```
中性估值 = EV 63.76 × PEV目标0.8 = 51.01元
安全边际 = (51.01 - 31.05) / 51.01 = 39.1%
```

## 逆向检查触发

Step 5总分=-40分（<-20人工复核门槛）。原因：6项轻微偏见叠加（确认偏见/锚定效应/社会认同/对比效应/易得性偏见/权威偏见），非基本面恶化。

**人工复核结论**：风险警示保留但不改变结论方向。PEV 0.49倍+NBV+40%+偿付能力216%支持"低估"判断，低PEV反映市场对利率的系统性担忧。

## 关键教训

1. **PEV是保险行业最准确的估值方法**：PE受投资收益波动影响大（2023年净利-27%），PB忽略业务价值，只有PEV同时包含净资产+未来利润贴现
2. **PEV<0.5不一定是买入信号**：日本保险公司PEV长期<0.5（利率趋零），需结合基本面判断
3. **行业校准至关重要**：若用标准PE分位评估，保险股PE 5-7倍看起来"合理"，但PEV 0.49倍才揭示真实低估程度
4. **逆向检查叠加扣分易触发上限**：6项轻微偏见即达-50分上限，需人工复核区分"偏见驱动"和"基本面驱动"
