### 图表输出规范（强制）

> **原则**：报告中必须使用图表增强可读性，特别是以下场景必须用图表：

**必须使用图表的场景**：

| 场景 | 图表类型 | 示例 |
|------|---------|------|
| 关键财务指标 | 仪表盘/卡片图 | 营收、净利、毛利率、ROE一目了然 |
| 近30天事件 | 时间线+影响力矩阵 | 事件按时间排列，影响力用颜色区分 |
| 周期定位 | 仪表盘/雷达图 | 经济周期、利率、通胀、流动性综合展示 |
| 行业景气度 | 趋势图 | 景气度随时间变化的曲线 |
| 产业链图谱 | 流程图 | 上游→中游→下游，标注公司位置 |
| 估值区间 | 区间图 | 保守/中性/乐观估值+当前价格 |
| 技术面 | 布林带图+均线图 | 价格在布林带中的位置 |
| 评分雷达 | 雷达图 | 企业质量/财务/产业链/估值各维度 |
| 五力模型 | 雷达图 | 五力强度可视化 |
| 护城河评估 | 矩阵图 | 护城河类型×宽度×趋势 |

**图表格式要求**：
1. 使用 ASCII 字符绘制（兼容纯文本环境）
2. 关键数据用框线突出
3. 颜色用 emoji 代替（🟢好/🟡一般/🔴差）
4. 图表下方必须有简要说明
