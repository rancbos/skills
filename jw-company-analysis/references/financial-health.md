> **输入**：step0_data.json → **输出**：step2_financial.json（2.1-2.4评分，供 Step 5/7 使用）

**核心公式快速参考**：
| 概念 | 公式 | 优秀标准 |
|------|------|----------|
| 核心利润 | 营收-成本-税金-销售-管理-研发-利息 | 核心利润率>10%，占营业利润>60% |
| 核心利润获现率 | 经营现金流/核心利润 | ≥1.0 |
| 资产金融负债率 | 金融性负债/资产总额 | <20%优秀，>40%警惕，>70%红线 |
| 资产有息负债率 | 有息负债/资产总额 | <30%优秀，≥50%偿债风险高 |
| 净商业信用占用 | 应付款项-应收款项 | 正值越大=竞争优势越显著 |
| 动态偿债年限 | 净金融负债/平均FCF(3年) | <2年很好 |

### 2.1 盈利质量（0-25分）

**🔴 利润三大前提过滤器**（进入详细分析前的强制前置检验）：
- **①利润为真**：经营现金流/净利润>1.0；应收账款增速<营收增速 → 不通过=**直接排除**
- **②利润可持续**：连续5年盈利；毛利率波动<3个百分点 → 不通过=降级（周期行业用"近10年≥7年盈利"替代）
- **③维持盈利不需大量资本**：FCF/净利润>80%；维护性资本开支<折旧 → 不通过=降级

| 指标 | 优秀标准 | 评分 |
|------|---------|------|
| 经营现金流/净利润 | >1.0 | 0-7分 |
| 应收账款增速/营收增速 | <1.0 | 0-6分 |
| 存货周转率趋势 | 波动<3pp或趋势向上 | 0-6分 |
| 非经常性损益占比 | <10% | 0-6分 |

**行业校准**：金融类经营现金流/净利润指标不适用，用拨备覆盖率替代；重资产制造类存货周转率标准放宽（行业均值×0.8）；周期品在景气低谷非经常性损益占比可能异常高，需用周期均值评估。

**会计诡计红旗**：EBITDA误导 / 重组"大洗澡" / 期权不计入费用 / 净利润操纵 / 退休金假设过高。

**📚盈利质量扩展**：详见 `references/investment-theory.md`

**毛利率×存货周转组合信号**：高毛利+高周转=🟢最佳；低毛利+高周转=⚪靠走量；高毛利+低周转=🟡需关注；双降=🔴根本性危机。

### 2.2 资产负债表健康度（0-25分）

| 指标 | 优秀标准（非金融） | 评分 |
|------|-----------------|------|
| 资产负债率 | <50% | 0-7分 |
| 有息负债率 | <30% | 0-6分 |
| 流动比率 | >1.5 | 0-6分 |
| 商誉/净资产 | <20% | 0-6分 |

**行业校准系数**（以行业中位数为基准）：24个申万一级行业校准系数表📚§39

**资产金融负债率**（张新民）：金融性负债/资产总额。<20%优秀，20-40%一般，>40%警惕。**70%红线**看此指标而非总负债率。（校准：基于A股非金融企业2015-2025年数据，高杠杆行业需用行业校准系数调整。）

**资产有息负债率**（张新民）：有息负债/资产总额。<30%优秀，30-50%一般，≥50%偿债风险高。（校准：基于A股非金融企业中位数，银行/保险不适用。）

**📚资产负债表扩展**：详见 `references/investment-theory.md`

**动态偿债年限**：净金融负债÷平均FCF(3年)，<2年很好/>5年麻烦。

### 2.3 现金流质量（0-25分）

| 指标 | 优秀标准 | 评分 |
|------|---------|------|
| 经营现金流持续性 | 连续5年为正 | 0-7分 |
| 自由现金流 | 连续3年为正 | 0-6分 |
| FCF/净利润 | >80%（资本密集型对比行业中位数） | 0-6分 |
| 所有者收益 | 连续3年增长或5年CAGR>行业中位数 | 0-6分 |

**行业校准**：金融类现金流指标不适用（用拨备覆盖率+资本充足率替代）；重资产制造类经营现金流/净利润标准放宽至>0.6（折旧占比高）；平台类自由现金流可能长期为负（投入期），需区分"烧钱增长"和"经营恶化"。

**所有者收益** = 净利润 + 折旧摊销 - 维护性资本开支。**股东盈余** = 净利润 + 折旧 + 摊销 - 资本支出。

**货币资金造假识别**：货币资金高+借款高+利息费用高（"三高并存"）=造假嫌疑。

**📚现金流扩展**：详见 `references/investment-theory.md`

### 2.4 财务趋势（0-25分）

**"一美元前提"检验**：市值增长/留存收益 > 1 = 资本配置优秀；< 1 = 不合格。

| 指标 | 优秀标准 | 评分 |
|------|---------|------|
| 5年ROE趋势 | 稳定高于行业中位数 | 0-8分 |
| 5年毛利率趋势 | 波动<3pp或趋势向上 | 0-6分 |
| 5年营收CAGR | >行业中位数 | 0-6分 |
| 5年净利润CAGR | >行业中位数 | 0-5分 |

**行业校准**：科技类ROE标准放宽至>8%（研发投入高）；周期品用10年均值评估5年趋势；金融类重点看净息差趋势而非ROE趋势；地产类营收CAGR在调控期可能为负，需区分"主动收缩"和"被动下滑"。

**趋势比绝对值更重要**：ROE从20%降到15%比一直10%更危险。

**📚财务趋势扩展**：详见 `references/investment-theory.md`

**财务健康总分 = 四项之和（满分100分）**
