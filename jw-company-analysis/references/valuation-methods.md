### 4.0 行业估值方法选择（强制前置）

> **⚠️ 核心问题**：Step 4.1 默认的"DCF+PE分位"组合对金融/周期/科技等行业严重失真。本节根据 Step 0.4.1 判定的 `industry_type`，自动选择行业专用估值方法，确保 Step 4 评分反映真实估值水平。

**执行规则**：
1. **读取 `industry_type`**：从 step0_data.json 的 `industry_type` 字段获取（Step 0.4.1 输出）
2. **匹配下表**：取对应行业的"主方法"+"辅方法"
3. **跳过不适用的方法**：若行业专用方法已覆盖（如保险用PEV），Step 4.1 中不适用的方法（如PE分位）直接跳过，不参与评分
4. **多方法加权**：主方法权重60%，辅方法权重40%（若有2个辅方法则各20%）

| 行业类型 | 主方法 | 辅方法 | 不适用方法 | 说明 |
|---------|--------|--------|-----------|------|
| **金融-银行** | PB分位 | DDM（股息折现） | PE分位、PEG、DCF | 银行利润受拨备调节大，PE失真；PB反映净资产质量 |
| **金融-保险** | PEV（内含价值倍数） | PB + DDM | PE分位、PEG、DCF | 保险利润波动大（投资收益），PEV含未来利润贴现 |
| **金融-券商** | PB+量能校准 | 正常化PE | DCF、PEG | 券商利润随市场波动剧烈，需量能信号修正PB |
| **金融-地产** | NAV（净资产价值） | PB + DCF | PE分位、PEG | 土地储备价值>账面净资产，NAV更真实 |
| **周期-资源** | 正常化PE（5-10年均值） | 产能市值比 + PB | 当期PE、PEG | 当期PE在周期顶部极低（陷阱），底部极高（误导） |
| **消费-品牌** | DCF所有者收益 | PE分位 + 股息率 | PS、PB | 现金流稳定，DCF最适合；PE分位辅助择时 |
| **科技-成长** | PEG + PS | DCF（如有正FCF） | PB、PE分位 | 高增长期利润低，PEG看增长性价比；无利润用PS |
| **医药-创新** | DCF+管线估值 | PS + PE分位（如有正利润） | PB | 管线价值=核心资产，传统PB/PE忽略管线 |
| **制造-传统** | DCF所有者收益 | PE分位 + PB | PS | 现金流可预测，DCF适用；PE/PB辅助验证 |
| **公用事业** | DDM（股息折现） | DCF + 股息率 | PEG、PS | 现金流极稳定，股息是核心回报来源 |

**PEV 估值法简述**（保险行业核心方法）：
```
PEV = 市值 / 内含价值（EV）
EV = 调整后净资产 + 有效业务价值（已签保单未来利润现值）
```
- PEV < 0.5：极度低估（中国太保2026年PEV 0.49倍）
- PEV 0.5-0.8：低估
- PEV 0.8-1.2：合理
- PEV > 1.5：高估
- **关键**：PEV隐含了未来增长预期，PEV<1意味着市场认为公司未来增长为负。实战案例详见 `references/insurance-pev-case-taiping.md`（中国太保：PEV 0.49倍+NBV+40%+偿付能力216%）。

**正常化PE估值法**（周期行业核心方法）：
```
正常化PE = 当前市值 / 近5-10年平均净利润
安全边际 = (正常化PE参考区间上限 - 当前正常化PE) / 正常化PE参考区间上限
```
- 周期行业正常化PE参考区间：钢铁8-15倍、煤炭6-12倍、有色金属10-18倍、航运8-15倍
- **关键**：在周期底部（利润低谷），正常化PE会远低于当期PE，此时反而是买入时机

**DDM估值法简述**（银行/公用事业核心方法）：
```
内在价值 = D₁ / (r - g)
D₁ = 预期下年每股股息
r = 要求回报率（10-12%）
g = 股息长期增长率（2-4%）
```
- 适用条件：股息政策稳定、增长率可预测、**分红率>30%**
- 银行/公用事业通常满足此条件
- **⚠️ DDM局限性**：分红率<25%时，DDM估值严重偏低（因D₁过小），仅作底线参考，不参与主估值判断。例如：保险公司分红率20%时，DDM可能给出远低于PEV/PB的估值，此时应以PEV/PB为准
- **多方法冲突处理**：若DDM与主方法（PEV/PB/DCF）估值差距>50%，以主方法为准，DDM仅标注为"底线参考"并在报告中说明冲突原因

**⚠️ 与 Step 4.1 的衔接**：
- Step 4.0 确定"用什么方法"→ Step 4.1 执行"具体计算"
- Step 4.1 的 DCF/PE分位/PEG等通用方法仍保留，但与本节方法叠加时：
  - 若行业专用方法与通用方法结论一致 → 安全边际加成+5分
  - 若结论冲突 → 以更保守结论为准，并在报告中说明冲突原因
  - 若行业专用方法不适用（如PEV所需EV数据不可得）→ 降级为PB+DCF，标注数据缺失
- **⚠️ 不适用方法处理规则（强制）**：Step 4.0 选择表中标注"不适用"的方法（如保险行业的PE分位/PEG/DCF），必须在报告中标注`[跳过：XX方法不适用于{行业类型}，原因：YY]`，**不参与Step 4.1评分**。未标注跳过而直接使用不适用方法→评分无效

### 4.1 内在价值估算（0-25分）

**首选：DCF 所有者收益法**（适用行业见 Step 4.0 选择表）：
```
内在价值 = Σ [所有者收益_t / (1+r)^t] + 终值
所有者收益 = 净利润 + 折旧摊销 - 维护性资本开支
```
DCF参数：折现率10-12%（A股建议11-12%），预测期5-10年，终值增长率2-3%。

**相对估值法（按公司类型选择，每次分析只选 2-3 种）**：

| 公司类型 | 首选方法 | 辅助方法 | 详见 |
|---------|---------|---------|------|
| 稳定盈利（茅台/可口可乐） | DCF所有者收益 + PE分位 | PB + 股息率 | §10, §37, §65 |
| 高增长（宁德/特斯拉） | PEG + DCF | PS + EV/EBITDA | §72-§73, §96, §139 |

### 4.1.1 PE分行业锚定区间

> PE不是万能尺子，不同行业合理PE天花板差异极大。用同一把尺子衡量所有行业是最常见的投资认知错误。

**PE核心公式**：
- `PE = 股价 ÷ EPS`
- `PE = 1 ÷ (ROE - g)`（ROE=盈利能力，g=增长率，r=贴现率）
- 利率下行→所有行业合理PE整体抬升；利率上行→PE天花板全面压缩

**PE本质由三件事决定**：ROE（盈利能力）、g（增长率）、r（贴现率/利率）。同一家公司，降息周期合理PE可以是30倍，加息周期可能只有18倍。**估值不是绝对值，是相对于利率的动态锚定。**

**各行业合理PE区间**：

| 行业类型 | 代表标的 | 合理PE区间 | 核心估值逻辑 |
|---------|---------|-----------|-------------|
| 金融（银行/保险/券商） | 工商银行、招商银行 | 5～15倍 | 分红逻辑，低弹性资产 |
| 地产 | 龙头房企 | 10～15倍（用PB更准） | 资产折价逻辑 |
| 强周期（钢铁/煤炭） | 宝钢股份、中国神华 | PE反向，看PB | 周期底部买顶部卖 |
| 食品饮料 | 贵州茅台、五粮液 | 15～25倍 | 稳定成长溢价，机构在PE触及25倍时兑现压力增大 |
| 农林牧渔/种业 | 牧原股份、隆平高科 | ≤25倍（谨慎） | 弱周期，类似周期逻辑 |
| 创新药/CXO | 药明康德、恒瑞医药 | 45～70倍 | 研发管线价值定价，龙头估值底线45倍 |
| 半导体/AI算力 | 中际旭创、北方华创 | 65～85倍 | 技术壁垒+市场空间定价，稀缺性>当期利润 |
| 次新股 | 上市1年内标的 | 同类老股1.5～2.5倍 | 筹码博弈情绪溢价 |

**科技股PE溢价合理性**：当一家公司拥有极高技术壁垒，市场给予远超行业均值的估值溢价，不是泡沫，而是对未来垄断利润的提前定价。科技股看的是：产业地位+技术卡位+市场空间。

**周期股PE反向逻辑**：利润在底部极低→分母小→PE虚高（实为买点）；利润在高位极高→分母大→PE虚低（实为顶部）。周期股更适合看PB，而不是PE。

### 4.1.2 PEG成长估值法（戴维斯双击）

**PEG核心公式**：`PEG = PE ÷ G`
- PE = 当前股价 ÷ 未来一年滚动EPS
- G = 未来一年盈利增速（去掉百分号的纯数字）
- **戴维斯双击** = 业绩增长 × 估值提升，两者叠加形成几何式上涨
  - 第一阶段（启动）：EPS翻倍(×2) × PE从20→40倍(×2) = 股价涨4倍
  - 第二阶段（加速）：EPS再翻倍(×2) × PE从40→60倍(×1.5) = 股价再涨3倍
  - 累计回报：4×3 = 12倍。涨幅是乘法不是加法，PEG是捕捉这种"业绩×估值"共振的量化工具

**未来一年EPS滚动折算**（关键细节）：

| 当前月份 | 计算公式 |
|---------|---------|
| 3月 | EPS = (10/12)×当年EPS + (2/12)×次年EPS |
| 6月 | EPS = (6/12)×当年EPS + (6/12)×次年EPS |
| 9月 | EPS = (3/12)×当年EPS + (9/12)×次年EPS |

⚠️ EPS数据来源：参考5家以上券商研报一致预期均值，避免单一来源预测误差。

**PEG买卖阈值（按市场量能动态调整，仅A股适用）**：

> **量能校准器**：2.5万亿日均成交额为宽松基准（详见4.1.3 PB估值法）。PEG阈值随量能浮动：<1.5万亿收严、≥2.5万亿放宽。

| 量能区间 | PEG买点 | PEG卖点 |
|---------|---------|---------|
| <1.5万亿 | PEG<0.7 | PEG≥1.5减仓 |
| 1.5～2万亿 | PEG<0.8 | PEG≥2.0卖出 |
| 2～2.5万亿 | PEG<1.0 | PEG≥2.5风险 |
| ≥2.5万亿 | PEG<1.2 | PEG≥3.0警惕 |

**PEG适用行业**：半导体、创新药/CXO、军工、锂电池、软件/云计算、消费电子、医疗保健

**PEG不适用行业**：地产（用PB）、银行/保险/券商（用PB）、钢铁/煤炭（用PB+周期位置）

**PEG六步操作流程**：
1. **行业适用性判断** → 确认有稳定研发投入且增速可预测，排除不适用行业
2. **量能环境判断** → 查当日及近期成交额均值，确定买卖阈值（2.5万亿宽松，1.5万亿以下收严）
3. **获取EPS预期** → 查询Wind/Choice等终端，**必须取5家以上机构一致预期均值**
4. **折算滚动EPS** → 按当前月份权重加权计算未来12个月滚动EPS（最易算错的一步）
5. **计算PE与G** → PE = 股价 ÷ 滚动EPS；G = (未来EPS增速) × 100，**G必须去掉百分号**
6. **对照阈值执行** → PEG < 买点线 → 分批买入；PEG > 卖点线 → 逐步减仓；不可单独依赖PEG

> **📚实战案例**：中际旭创PEG计算、寒武纪动态视角警示等详见 `references/valuation-case-studies.md`

**成长股五类分型**：详见 4.1.6 科技股五类分型与估值策略。

### 4.1.3 PB资产估值法

**PB核心公式**：`PB = 股价 ÷ 每股净资产 = 总市值 ÷ 净资产`

**PB适用场景**：当期利润被外部因素严重扭曲时（拨备计提、投资浮盈亏、结转周期、商品价格极值），资产本身才是真实价值载体。盈利只是资产利用率的结果，必须用PB定价。

**六大适用行业**：银行、保险、券商、地产、钢铁、煤炭（合计占A股1/3以上市值）

**PB与ROE的核心关系**：`合理PB = (ROE - g) ÷ (r - g)`
- ROE↑ → PB↑（盈利能力越强，账面资产越值钱）
- 利率↓ → PB↑（贴现率下降，资产重估上行）
- g预期↓ → PB压缩（预期利润下行，估值收缩）

**量能对PB估值的系统性影响**（外部校准器）：

| 量能水平（日均成交额） | 对PB的影响与实战含义 |
|----------------------|-------------------|
| < 1.5万亿 | 流动性显著收缩，资产折价压力极大。低PB不代表见底，应以观望为主 |
| 1.5-2.0万亿 | 正常运行区间。历史中枢PB指标恢复适用，按行业历史均值PB判断 |
| 2.0-2.5万亿 | 流动性明显改善，资产估值进入修复期。PB接近历史中枢时可积极关注 |
| ≥2.5万亿 | 宽松基准。流动性充裕，资产估值溢价整体向上平移。PB低于历史均值即为积极布局信号 |

> **关键结论**：在2.5万亿基准下，历史底部PB的支撑力大幅增强。但若量能萎缩至1.5万亿以下，即使PB看起来极低也不要轻易"抄底"——流动性不足导致的资产折价可以持续非常久。

**六大行业PB区间速查**（详细分析见 `references/industry-pb-details.md`）：

| 行业 | PB核心区间 | 关键指标 | 特殊警示 |
|------|-----------|---------|---------|
| 银行 | 0.6~0.8甜蜜区 | 不良率/拨备覆盖率/ROE | PB<0.5查资质，>1.0警惕 |
| 地产 | 0.4~0.6深度折价 | 有息负债/土储/国资背书 | 民企PB<0.4先排雷 |
| 保险 | 0.8~1.2合理中枢 | P/EV(内含价值) | PB跟着利率走 |
| 券商 | 量能决定PB | 成交额×两融×IPO | PE极具欺骗性，必须用PB |
| 钢铁 | 0.8~1.5景气修复 | 周期位置/产能利用率 | PB>1.5+PE极低=顶部陷阱 |
| 煤炭 | <0.8周期底部 | 股息率>5%/分红政策 | 三先决条件必须同时满足 |

**量能校准器**（2.5万亿为宽松基准）：<1.5万亿观望/1.5-2万亿正常/2-2.5万亿积极/≥2.5万亿宽松。

**PB投资四步框架**：①确认行业（是否资产驱动型）→ ②判断周期位置 → ③看PB是否低于历史中枢 → ④确认ROE未崩、行业非永久衰退

**终极判断**：低PB + ROE未崩 + 非结构衰退 = 中长期布局区间；低PB + ROE崩塌 + 结构性衰退 = 坚决规避的价值陷阱

**PB三大常见误区与避坑**：

| # | 误区 | 正确做法 |
|---|------|---------|
| 1 | PB低于1就是低估可以买 | 破净只是触发关注的信号，买入前必须验证净资产质量、债务情况和ROE趋势 |
| 2 | 只看PB绝对值高低 | 必须结合行业周期位置。景气下行期的0.8倍可能还会跌向0.5倍 |
| 3 | 给科技股套用PB估值 | 科技公司的核心是人才、专利和数据，必须用PE/PEG/PS/DCF，切勿套用PB |

**核心结论**：用错估值工具，比没有工具更危险。明确行业的资产驱动属性，是用好PB的绝对前提。

### 4.1.4 PS市销率估值法

**PS核心公式**：`PS = 总市值 ÷ 年度营业收入 = 股价 ÷ 每股营收`

**PS适用场景**（PE失效时的替代工具，定位为"最后一道防线"）：

| PE失效场景 | 典型案例 | 核心失效原因 |
|-----------|---------|------------|
| 当期大幅亏损 | 寒武纪 | 分母为负，PE无法计算 |
| 利润极度微薄 | 成长期科技公司 | PE高达千倍，失去比较意义 |
| 利润一次性扭曲 | 政府补贴/资产出售 | 剔除一次性项目后可能由盈转亏 |
| 研发密集型 | 早期AI、生物医药 | 战略性研发投入压低利润，亏损是"投资未来" |
| 商业模式重构 | 互联网平台初期 | 亏损是主动换取用户规模，关注营收比利润更重要 |

**估值工具选择优先级**：能用PE就用PE → PE失效用PEG → 现金流稳定用DCF → **PS是最后一道防线**，当前面三个都失效时才登场。PS不是更好的工具，而是PE失效后的替代工具。

**PS两大驱动因素**：毛利率 × 增速

| 毛利率 | 增速 | PS参考区间 | 典型行业 |
|--------|------|-----------|---------|
| 高 | 高 | 20～100倍 | AI软件、SaaS龙头 |
| 高 | 低 | 6～12倍 | 成熟软件、医药 |
| 低 | 高 | 1～3倍 | 电商、硬件组装 |
| 低 | 低 | <1倍 | 传统制造、低端零售 |

**PS行业中枢参考**（2025年，2.5万亿量能背景）：AI算力稀缺标的的高PS包含"稀缺性+成长叙事"双重溢价。当稀缺性下降（如竞品上市），PS会快速回归行业均值。详见 `references/valuation-case-studies.md`

**PS行业参考区间口诀**：
| 组合特征 | 典型行业 | PS参考 |
|---------|---------|--------|
| 毛利80%+增速100% | AI软件/SaaS | 50~100倍 |
| 毛利60%+增速50% | 芯片设计/医疗 | 20~40倍 |
| 毛利40%+增速20% | 硬件/应用 | 6~12倍 |
| 毛利20%+增速10% | 组装/零售 | <2倍 |

**PS四大误区**：
1. ❌ PS越低越好 → ✅ 先看毛利率和增速
2. ❌ 低毛利行业用PS → ✅ 直接用PE或PB
3. ❌ 忽视增速可持续性 → ✅ 拆解营收结构
4. ❌ 低增速+低PS=机会 → ✅ 可能是价值陷阱

**PS铁律**：不看毛利率就谈PS高低是无效分析。两家公司PS都是20倍，高毛利那家可能便宜，低毛利那家可能贵出天际。

> **📚实战案例**：科大讯飞PS四步合理性判断法详见 `references/valuation-case-studies.md`

**"市概率"定价模型**（适用于未盈利AI/科技企业）：
- 隐含市值 = 成熟龙头市值 × 成功概率(P)
- 锚定对象：对标全球龙头（如寒武纪→英伟达20万亿市值）
- 概率定价推演：P=1%→2000亿；P=3%→6000亿；P=5%→10000亿
- **本质是概率赌注，不是传统价值投资**。灵魂拷问：如果叙事失败，市值底在哪里？

**PS适用边界明确化**：
- ✅ 适合：尚未盈利但高增长（50%+）/ 研发压薄利润（净利率<3%）/ 跨市场横向比较 / 周期底部亏损
- ❌ 不适合：稳定盈利成熟企业（用PE）/ 资产驱动行业（用PB）/ 极低毛利率行业（PS0.5倍都可能贵）/ 早期概念公司（营收<1亿）

**PS实操清单**：
1. 明确毛利率量级（软件 vs 硬件）
2. 验证增速可持续性（订单拆解，区分订阅型高质量 vs 项目型低质量）
3. 设定盈利接力时间表（何时从PS定价切换回PE定价）
4. 设止损与悲观情景（底线思维：如果叙事失败，市值底在哪里？）

### 4.1.5 估值工具选择地图

**三句话掌握全部估值逻辑**：
1. **低增速行业** → 看PE上限，超过锚定区间即风险溢价不足
2. **强周期/资产型行业** → 看PB和周期位置，底部折价买入
3. **高成长科技行业** → 看PEG和技术壁垒，结合流动性环境动态判断

**完整估值方法对照表**：

| 行业类型 | 首选方法 | 辅助指标 | 核心判断逻辑 |
|---------|---------|---------|-------------|
| 金融（银行/保险/券商） | PB | ROE/不良率 | ROE决定PB合理中枢 |
| 地产 | PB | NAV/土储 | 净资产折价程度 |
| 强周期（钢铁/煤炭） | PB | 现金流/分红率 | 周期底部折价程度 |
| 食品饮料/消费 | PE | 净利润增速 | 25倍以下合理配置区 |
| 成长科技/医药 | PEG | PE/增速G | PEG<1低估，≥2卖出 |
| 大市值成熟科技 | DCF | 自由现金流 | 未来10年现金流贴现 |
| 未盈利科技 | PS | 烧钱率/营收增速 | 成长潜力替代当期利润 |

**进阶认知：利率对估值的全局影响**：

利率环境是估值判断的底层逻辑（PE=1÷(ROE-g)，详见4.1.1）。同一家公司，降息周期合理PE可以是30倍，加息周期可能只有18倍。估值不是绝对值，是相对于利率的动态锚定。

| 宏观环境 | 对估值的影响 |
|---------|------------|
| 利率下行周期 | 所有行业合理PE整体抬升（分母变小） |
| 利率上行周期 | 所有行业PE天花板全面压缩（分母变大） |
| 经济扩张期 | 成长股g预期提升，PEG更宽松 |
| 经济收缩期 | 防御性资产（高股息）相对占优 |

**量能作为估值外部校准器**（统一原则）：详见4.1.3 PB估值法中的"量能对PB估值的系统性影响"表格。2.5万亿为宽松基准，1.5万亿以下观望。**核心结论**：无量不谈估值。

**估值方法选择口诀**（课件总结）：
> 周期股看PE容易被骗，金融股看PE永远看不明白。真正理解资产行业，必须理解PB。真正理解成长行业，必须理解PEG与戴维斯双击。

**估值工具选择快速参考**：

| 工具 | 核心适用场景 | 关键前提 | 失效条件 |
|------|------------|---------|---------|
| PE | 稳定盈利/消费/公用事业 | 利润稳定且为正 | 亏损/利润大幅波动 |
| PEG | 高成长科技/创新药/AI应用 | 有增速+有盈利 | 零增速/亏损 |
| PB | 银行/地产/钢铁/保险 | 资产驱动型行业 | 轻资产/纯科技 |
| DCF | 水电/高速/优质消费 | 现金流可预测 | 高成长期假设误差过大 |
| PS | 亏损科技/高成长/国产替代 | 营收真实可信 | 零增速/低毛利 |

**黄金法则**：工具不是越高级越好，而是越匹配越好。选对工具是估值分析的第一步，也是最重要的一步。

### 4.1.6 科技股五类分型与估值策略

> 科技股并非铁板一块。不同市值规模、盈利状态、产业地位的科技公司，对应完全不同的估值方法和仓位管理。

| 类型 | 特征 | 估值方法 | 操作策略 | 仓位建议 |
|------|------|---------|---------|---------|
| **老牌龙头** | 大市值（沪深300成份）、行业绝对第一 | PEG + DCF | 长期价值配置，追求确定性收益 | 核心仓 |
| **挑战型成长** | 200亿~2000亿，有望冲击龙头 | PEG<2 | 右侧交易，月K/周K趋势确认后介入 | 积极配置 |
| **小市值高速** | <150亿，CAGR 30%~50%，有技术有产品 | PEG辅助 | 短周期波段，增速滑落即刻止损 | 适度参与 |
| **未盈利爆发** | 当前亏损EPS为负，产业方向有爆发潜力 | 烧钱率/营收增速 | 长期跟踪，等待产品落地引爆 | ≤10%博弈仓 |
| **价值崩塌** | 主营萎缩、技术替代、趋势断头 | 无投资价值 | 坚决回避，第一时间止损离场 | 0% |

**未盈利爆发型四维判断**：
- **烧钱率**：每次融资后烧完的时间是否在拉长？（烧钱速度降低=营收开始填补亏损）
- **营收增速**：是否每年持续高速增长？（维持50%+年增速为佳）
- **经营现金流**：是否维持正常？（现金流健康则企业不会破产）
- **行业地位**：是否为细分领域全球/国内龙头？（龙头一旦引爆，爆发力最强）

**价值崩塌型识别信号**：大厂打价格战→毛利率持续下滑 / 核心产品被新技术路线替代 / 股价跌破重要技术支撑。**投资原则**：产业阵痛期间没有任何投资价值，不要越跌越买，等待新产品上线后再重新评估。

**📚估值理论扩展**：详见 `references/investment-theory.md` / `jw-company-valuation` skill

**估值输出原则**：内在价值是区间，非精确数字。「宁要模糊的正确，不要精确的错误。」估值区间宽度按公司类型差异化：
- **稳定盈利**（茅台/可口可乐）：±15-20%
- **高增长**（宁德/特斯拉）：±30-50%
- **周期性**（钢铁/航运）：±40-60%
- **重资产**（地产/基建）：±20-30%
- **亏损有收入**：±50-80%

**执行指引**：①用DCF/PE/PB等方法算出中性估值→②按公司类型确定区间宽度→③区间下限=建仓上限，区间上限=减仓参考→④安全边际从区间下限计算（如中性估值100元±20%，区间80-120元，安全边际30%→建仓价56元）。

**评分标准**：21-25分（DCF+相对估值交叉验证一致）/ 16-20分（之一可靠）/ 11-15分（有不确定性）/ 0-10分（估值困难）

### 4.2 安全边际评估（0-25分）

```
安全边际 = (内在价值 - 当前价格) / 内在价值
```

| 安全边际 | 评分 | 含义 |
|---------|------|------|
| > 40% | 21-25分 | 极佳：大幅低估 |
| 25-40% | 15-20分 | 良好：明显低估 |
| 10-25% | 10-14分 | 一般：略有低估 |
| < 10% | 0-9分 | 不足：接近合理价值 |

**A股安全边际校准**：散户占比高、情绪波动大，建议优秀企业>30%，一般企业>50%。🇨🇳详见china-value-investing.md。

**📚安全边际扩展**：详见 `references/investment-theory.md` / `references/china-value-investing.md`

### 4.2.1 估值结合因素

> 估值不是孤立的数字判断，必须结合四个维度综合评估。

**①公司历史估值区间**：当前PE/PB处于公司过去5年估值区间的什么位置？（<25%分位=低估/25-75%=正常/>75%=高估）。**注意**：历史区间仅作参考，不能机械套用——公司基本面变化时，合理估值区间也会变化。

**②同行业可比公司估值**：当前估值相对同行是偏高还是偏低？偏高的原因是什么（护城河更强/增长更快/确定性更高）？偏低的原因是什么（基本面差/市场偏见/流动性差）？

**③当前景气周期位置**：结合 Step 0.6 宏观环境和 Step 3.0 行业阶段，判断当前处于周期哪个位置。周期底部估值偏高是正常的（利润被压低），周期顶部估值偏低也是正常的（利润虚高）。

**④未来盈利增速与确定性**：增速越高，可接受的估值越高；确定性越强，可接受的估值越高。**增速×确定性**是估值溢价的核心逻辑。

**估值综合判断公式**：合理估值 = 基础估值 × (1 + 增速溢价) × (1 + 确定性溢价) × (1 + 周期位置调整)

### 4.3 估值陷阱识别（0-25分）

满分25分，每触发一项扣5分，最低0分。

| 陷阱类型 | 触发条件 | 扣分 |
|---------|---------|------|
| 低PE陷阱 | PE低于行业中位数50%以上，但基本面恶化（营收/利润连续下滑） | -5分 |
| 高增长陷阱 | 过去3年CAGR>30%，但增长不可持续（市场饱和/竞争加剧/政策退坡） | -5分 |
| 资产价值陷阱 | 账面资产丰厚但变现困难（大量商誉/在建工程/长期应收款） | -5分 |
| 股息陷阱 | 当前股息率>5%，但不可持续（利润下滑/负债率上升/分红率>100%） | -5分 |
| 商誉陷阱 | 商誉/净资产>30%，且被并购标的业绩承诺期临近结束 | -5分 |

**行业校准**：周期品在景气高点PE极低但属正常，不触发低PE陷阱；地产类商誉占比通常较低，重点看资产价值陷阱；科技类高增长正常，需区分"真增长"和"靠并购堆增长"。

### 4.3.1 高估值合理性判断

> 高估值不一定贵，低估值不一定便宜。关键看高估值背后是否有合理支撑。

**高估值合理的三种情况**：

| 情况 | 判断标准 | 估值容忍度 |
|------|---------|-----------|
| 增长快且确定性高 | 未来3年CAGR>30% + 业绩兑现率>80% | PE可接受同行1.5-2倍 |
| 护城河极强 | 宽护城河+趋势加深+行业龙头 | PE可接受同行1.2-1.5倍 |
| 未来利润释放空间大 | 当前净利率<5%但趋势向上+规模效应显现 | 可用PS/PEG替代PE |

**高估值不合理的三种情况**：

| 情况 | 判断标准 | 风险 |
|------|---------|------|
| 增长不可持续 | 3年CAGR>50%但增速在放缓+市场饱和 | 戴维斯双杀 |
| 情绪驱动 | 鸡尾酒会理论第四阶段（出租车司机荐股） | 泡沫破裂 |
| 概念炒作 | 无业绩支撑的题材股（如纯概念股） | 回归基本面 |

**核心结论**：估值高低是相对的，必须对应增速、确定性和护城河强度来判断。PE 50倍的茅台可能比PE 10倍的地产更便宜——因为前者增长确定，后者可能在价值陷阱里。

**估值总分 = 三项之和（满分100分）**
