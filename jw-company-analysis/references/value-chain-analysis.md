# 产业链/价值链/波特五力分析框架

> 来源：波特五力模型、产业链分析、巴菲特"经济特许权"思想

## 一、产业链分析

### 产业链环节划分

| 环节 | 特征 | 附加值 | 典型行业 |
|------|------|--------|---------|
| **上游原材料** | 资源型、周期性强 | 中-低 | 矿业、农业、基础化工 |
| **中游制造** | 技术型、规模效应 | 中 | 电子制造、精细化工、零部件 |
| **下游品牌** | 品牌型、消费属性 | 高 | 奢侈品、消费电子、品牌食品 |
| **终端渠道** | 渠道型、网络效应 | 中-高 | 零售、电商、物流 |

### 微笑曲线

```
附加值
  ↑
  │   研发/品牌          营销/服务
  │      ╲                ╱
  │       ╲              ╱
  │        ╲            ╱
  │         ╲──────────╱
  │           制造/组装
  └──────────────────────────→ 产业链位置
```

**含义**：
- 研发/品牌和营销/服务附加值最高
- 制造/组装附加值最低
- 企业应尽量向高附加值环节移动

### 产业链定位评估

**评估维度**：
1. 处于产业链的哪个环节？
2. 上下游集中度如何？
3. 是否处于"微笑曲线"的高附加值环节？
4. 是否有向上游/下游延伸的能力？

**评分标准**：
- 高附加值环节+上下游分散：优秀
- 中等附加值环节：一般
- 低附加值环节+上下游集中：较差

## 二、波特五力模型

### 五力详解

#### 1. 现有竞争者的竞争强度

**影响因素**：
- 行业集中度（CR3/CR5/CR10）
- 产品差异化程度
- 固定成本高低
- 退出壁垒高低
- 行业增速

**有利信号**：
- CR3 > 60%（高度集中）
- 产品高度差异化
- 固定成本低
- 退出壁垒低
- 行业增速快

**不利信号**：
- 行业分散（CR3 < 30%）
- 产品同质化
- 固定成本高
- 退出壁垒高
- 行业增速慢

#### 2. 潜在进入者的威胁

**进入壁垒**：
- 规模经济
- 资本需求
- 转换成本
- 产品差异化
- 分销渠道
- 政策壁垒（牌照、专利）

**有利信号**：
- 进入壁垒高
- 需要大量资本
- 需要技术积累
- 需要牌照/专利
- 需要品牌积累

**不利信号**：
- 进入壁垒低
- 资本需求小
- 技术门槛低
- 无牌照/专利保护
- 品牌壁垒弱

#### 3. 替代品的威胁

**替代品类型**：
- 功能替代（如电子书替代纸质书）
- 技术替代（如电动车替代燃油车）
- 模式替代（如在线教育替代线下教育）

**有利信号**：
- 无明显替代品
- 替代品性价比不高
- 转换成本高

**不利信号**：
- 替代品性价比更高
- 替代品技术进步快
- 转换成本低

#### 4. 供应商的议价能力

**议价能力来源**：
- 供应商集中度高
- 供应商产品差异化
- 供应商前向一体化威胁
- 转换供应商成本高

**有利信号**：
- 供应商分散
- 供应商产品同质化
- 供应商无前向一体化威胁
- 转换供应商成本低

**不利信号**：
- 供应商集中（如芯片供应商）
- 供应商产品差异化
- 供应商有前向一体化威胁
- 转换供应商成本高

#### 5. 购买者的议价能力

**议价能力来源**：
- 购买者集中度高
- 购买者购买量大
- 产品同质化
- 购买者后向一体化威胁
- 转换成本低

**有利信号**：
- 购买者分散
- 购买者购买量小
- 产品差异化
- 购买者无后向一体化威胁
- 转换成本高

**不利信号**：
- 购买者集中（如大客户依赖）
- 购买者购买量大
- 产品同质化
- 购买者有后向一体化威胁
- 转换成本低

### 五力综合评估

| 力量 | 有利 | 中性 | 不利 |
|------|------|------|------|
| 现有竞争 | 高集中+差异化 | 中等集中 | 分散+同质化 |
| 潜在进入 | 高壁垒 | 中等壁垒 | 低壁垒 |
| 替代品 | 无替代 | 有替代但性价比低 | 替代品更优 |
| 供应商议价 | 供应商分散 | 中等集中 | 供应商垄断 |
| 购买者议价 | 客户分散 | 中等集中 | 大客户依赖 |

**综合评分**：
- 五力全面有利：优秀
- 四力有利：良好
- 三力有利：一般
- 两力及以下有利：较差

## 三、议价能力分析

### 对上游的议价能力

**评估维度**：
- 供应商是否可替代？
- 原材料是否依赖单一来源？
- 是否有长期采购协议？
- 是否有前向一体化能力？

**强议价能力表现**：
- 供应商分散
- 原材料可替代
- 有长期采购协议
- 有前向一体化能力

**弱议价能力表现**：
- 供应商集中
- 原材料依赖单一来源
- 无长期采购协议
- 无前向一体化能力

### 对下游的议价能力

**评估维度**：
- 客户集中度如何？
- 是否能转嫁成本？
- 是否有定价权？
- 是否有后向一体化能力？

**强议价能力表现**：
- 客户分散
- 能转嫁成本
- 有定价权
- 有后向一体化能力

**弱议价能力表现**：
- 客户集中
- 不能转嫁成本
- 无定价权
- 无后向一体化能力

## 四、产业链风险分析

### 供应链风险

**单点故障风险**：
- 关键原材料依赖单一供应商
- 关键技术依赖单一来源
- 关键设备依赖单一供应商

**地缘政治风险**：
- 芯片供应链风险
- 稀土供应链风险
- 能源供应链风险

**自然灾害风险**：
- 地震、洪水等自然灾害
- 疫情等公共卫生事件
- 气候变化影响

### 技术风险

**技术替代风险**：
- 新技术可能替代现有技术
- 技术迭代速度快
- 研发投入不足

**技术依赖风险**：
- 核心技术依赖外部
- 专利到期风险
- 技术泄露风险

### 政策风险

**行业政策风险**：
- 行业监管政策变化
- 行业补贴政策变化
- 行业准入政策变化

**贸易政策风险**：
- 关税政策变化
- 出口管制政策变化
- 贸易摩擦风险

## 五、经济特许权分析

### 巴菲特"经济特许权"定义

**经济特许权**：
- 产品或服务有需求
- 产品或服务不可替代
- 产品或服务有定价权

**经济特许权 vs 一般企业**：

| 维度 | 经济特许权 | 一般企业 |
|------|----------|---------|
| 需求 | 刚需 | 可选 |
| 替代性 | 不可替代 | 可替代 |
| 定价权 | 有定价权 | 无定价权 |
| 护城河 | 宽护城河 | 窄护城河 |
| 盈利能力 | 高且稳定 | 低且波动 |

### 经济特许权评估

**评估维度**：
1. 产品是否有刚需？
2. 产品是否不可替代？
3. 是否有定价权？
4. 客户是否愿意支付溢价？
5. 即使涨价客户是否也不走？

**评分标准**：
- 五维度全面优秀：经济特许权
- 三维度优秀：有一定特许权
- 两维度以下：无特许权

## 六、产业链趋势分析

### 产业链重构趋势

**新能源替代传统能源**：
- 光伏、风电替代火电
- 电动车替代燃油车
- 储能技术发展

**国产替代/自主可控**：
- 芯片国产替代
- 软件国产替代
- 设备国产替代

**数字化转型**：
- 传统行业数字化
- 云计算、大数据
- 人工智能应用

### 产业集中度变化

**整合趋势**：
- 行业集中度提升
- 龙头企业份额扩大
- 中小企业退出

**分散趋势**：
- 新进入者增多
- 技术门槛降低
- 市场细分化

### 产业链趋势评估

**评估维度**：
- 产业链是否在重构？
- 公司处于受益还是受损位置？
- 产业集中度在提升还是下降？
- 公司是否有能力适应变化？

**评分标准**：
- 趋势有利+公司处于受益位置：优秀
- 趋势中性：一般
- 趋势不利+公司处于受损位置：较差

## Step 3：行业赛道与产业链分析（权重 10%）

> **输入**：step0_data.json（行业/竞争对手数据）→ **输出**：step3_industry.json（3.0-3.4评分，供 Step 7 使用）
> 🔗详见value-chain-analysis.md

### 3.0 行业赛道评估（前置判断，定性评估，不参与产业链总分加权）

> **核心问题**：在分析公司之前，先判断它所在的行业是不是一个好赛道。好赛道里的一般公司，可能比差赛道里的优秀公司更值得投资。
>
> **简化评估**（3个子项，定性判断为主）：评估结果写入 step3_industry.json 的 `track_assessment` 字段，用于 Step 3 分析背景和 Step 7 综合判断。

**3.0.1 市场空间与阶段（定性）**

| 维度 | 评估要点 |
|------|---------|
| 市场规模 | 行业总规模（当前+未来3-5年预期）、渗透率（<30%有空间，>70%天花板） |
| 行业阶段 | 导入期（高风险高回报）→成长期（优选龙头）→成熟期（高分红+护城河）→衰退期（回避） |
| 竞争性质 | 增量竞争（行业变大）vs 存量竞争（零和博弈） |
| 景气度 | 当前上行/平稳/下行（结合PMI、产能利用率、库存周期判断） |

**3.0.2 竞争格局（定性）**

| 维度 | 评估要点 |
|------|---------|
| 集中度 | CR3>60%=高集中度有利龙头；分散竞争=格局不稳 |
| 玩家分布 | TOP5市占率、"赢家通吃"vs"分散竞争" |
| 进入壁垒 | 牌照/技术/品牌/规模/资本壁垒 |

**3.0.3 驱动因素与风险（定性）**

| 维度 | 评估要点 |
|------|---------|
| 核心驱动（3-5个） | 需求增长/技术升级/产品迭代/成本下降/替代效应/政策支持，各因素↑/→/↓ |
| 主要风险（必评5类） | 周期性波动/原材料价格/技术路线切换/监管收紧/需求不及预期 |

**赛道评级**：优质赛道（多因素向上+风险可控）→ 正常赛道（结构性机会）→ 一般赛道（需精选个股）→ 差赛道（谨慎或回避）

### 3.1 产业链定位（0-25分）

产业链环节：上游原材料（中-低附加值）→ 中游制造（中）→ 下游品牌（高）→ 终端渠道（中-高）。处于"微笑曲线"高附加值环节加分。

**评分标准**：
- 20-25分：处于产业链核心环节（品牌/技术/平台），附加值高，上下游依赖度低
- 15-19分：处于产业链重要环节，有一定附加值，但面临上下游挤压
- 10-14分：处于产业链一般环节，附加值中等，议价空间有限
- 0-9分：处于产业链低端环节（纯代工/原材料），附加值低，易被替代

**行业校准**：金融/平台类企业不适用产业链定位评分，按商业模式评分替代；消费品重点关注品牌+渠道环节；科技类重点关注技术壁垒环节。

**📚产业链扩展**：详见 `references/investment-theory.md` / `references/value-chain-analysis.md`

### 3.2 议价能力（0-25分）

**波特五力评估**：现有竞争强度（CR3>60%有利）/ 潜在进入者威胁 / 替代品威胁 / 供应商议价能力 / 购买者议价能力。



**评分标准**：
- 20-25分：五力全部有利（CR3>60%、进入壁垒高、无替代品、供应商/购买者议价能力弱）
- 15-19分：五力中4项有利，1项中性
- 10-14分：五力中3项有利，2项不利
- 0-9分：五力中2项以上不利，竞争激烈或上下游议价能力强

**行业校准**：金融类用资本充足率+净息差替代波特五力；平台类重点关注网络效应+转换成本；周期品在行业低谷期五力全面恶化，需用周期中位数评估。

### 3.3 产业链风险（0-25分）

**评估维度**：供应链单点故障风险 / 关键原材料/技术受制于人风险 / 地缘政治影响 / 供应商集中度 / 替代技术威胁。

**评分标准**：
- 20-25分：供应链多元分散，无单点故障，关键原材料/技术自主可控，无地缘政治风险
- 15-19分：供应链基本稳定，少数环节有依赖但有备选方案
- 10-14分：存在一定供应链风险，部分关键环节受制于人
- 0-9分：供应链高度集中，关键环节严重依赖外部，地缘政治风险显著

### 3.4 产业链趋势（0-25分）

**评估维度**：产业链重构方向 / 国产替代进程 / 行业集中度变化 / 技术迭代趋势 / 政策驱动的产业链转移。

**评分标准**：
- 20-25分：公司处于产业链上升环节，受益于国产替代/技术升级/集中度提升等趋势
- 15-19分：产业链趋势对公司中性偏正面，无明显不利变化
- 10-14分：产业链存在一定不确定性，趋势方向不明确
- 0-9分：产业链趋势对公司不利，面临被替代/边缘化/产能过剩风险

**竞争性毁灭**（芒格）：关注行业竞争格局变化，优选具有持久竞争优势的企业。

**产业链总分 = 3.1+3.2+3.3+3.4 四项之和（满分100分）**。3.0 行业赛道评估为前置判断，其结论用于 Step 3 分析背景，但**不参与产业链总分加权**。
