# 技术分析框架

> 来源：技术分析理论、价值投资者技术面应用

## 一、定位说明

**技术分析在本框架中的定位**：
- 不是决策核心，而是"买什么"确定之后的"什么时候买"辅助工具
- 价值投资者用技术面优化入场时机，而非用技术面选股
- 技术面不参与加权总分的直接计算，而是作为建仓时机调整因子

---

## 二、均线系统

### 均线类型

| 均线 | 周期 | 含义 | 用途 |
|------|------|------|------|
| MA5 | 5日 | 周线 | 短期趋势 |
| MA10 | 10日 | 双周线 | 短期趋势 |
| MA20 | 20日 | 月线 | 短期趋势 |
| MA60 | 60日 | 季线 | 中期趋势 |
| MA120 | 120日 | 半年线 | 中长期趋势 |
| MA250 | 250日 | 年线 | 长期趋势 |

### 均线排列

**多头排列**：
- MA5 > MA10 > MA20 > MA60 > MA120 > MA250
- 含义：趋势向上，强势
- 操作：持有或买入

**空头排列**：
- MA5 < MA10 < MA20 < MA60 < MA120 < MA250
- 含义：趋势向下，弱势
- 操作：卖出或观望

**均线纠缠**：
- 多条均线粘合
- 含义：方向不明，变盘在即
- 操作：等待方向明确

### 价值投资者的均线用法

**核心用法**：
- 不追涨停板，等回踩MA20/MA60时建仓
- MA250（年线）是"牛熊分界线"
- 均线纠缠（多条均线粘合）→ 变盘信号，结合基本面决定方向

**建仓时机**：
- 基本面优秀 + 价格回踩MA20/MA60 = 建仓机会
- 基本面优秀 + 价格站稳MA250 = 基本面确认
- 基本面恶化 + 价格跌破MA250 = 卖出信号

## 三、MACD

### MACD构成

**DIF（快线）**：
- DIF = EMA12 - EMA26
- 反映短期趋势

**DEA（慢线）**：
- DEA = DIF的9日EMA
- 反映中期趋势

**MACD柱**：
- MACD = 2 × (DIF - DEA)
- 反映趋势强度

### MACD信号

**金叉**：
- DIF上穿DEA
- 含义：短期动能转强
- 操作：买入信号

**死叉**：
- DIF下穿DEA
- 含义：短期动能转弱
- 操作：卖出信号

**零轴之上**：
- DIF > 0
- 含义：中期趋势偏多
- 操作：持有或买入

**零轴之下**：
- DIF < 0
- 含义：中期趋势偏空
- 操作：卖出或观望

**底背离**：
- 价格新低但MACD不新低
- 含义：下跌动能衰竭，可能反转
- 操作：关注买入机会

**顶背离**：
- 价格新高但MACD不新高
- 含义：上涨动能衰竭，可能见顶
- 操作：关注卖出机会

### 价值投资者的MACD用法

**核心用法**：
- 底背离 + 基本面优秀 = 极佳买入时机
- 顶背离时不必急着卖（如果基本面没变），但需提高警惕
- 金叉 + 基本面优秀 = 买入确认
- 死叉 + 基本面恶化 = 卖出确认

## 四、RSI相对强弱指标

### RSI计算

```
RSI = 100 - 100/(1+RS)
RS = 平均上涨幅度 / 平均下跌幅度
```

### RSI区间

| RSI区间 | 含义 | 价值投资者视角 |
|---------|------|--------------|
| RSI < 20 | 极度超卖 | 🟢 如果基本面OK，可能是黄金坑 |
| RSI 20-30 | 超卖 | 🟢 关注是否出现止跌信号 |
| RSI 30-50 | 偏弱 | ⚪ 正常波动区间 |
| RSI 50-70 | 偏强 | ⚪ 正常波动区间 |
| RSI 70-80 | 超买 | 🟡 谨慎，考虑减仓（如果估值也高） |
| RSI > 80 | 极度超买 | 🔴 高估+超买=风险叠加 |

### 价值投资者的RSI用法

**核心用法**：
- RSI < 30 + 基本面优秀 = 买入机会
- RSI > 70 + 估值偏高 = 考虑减仓
- RSI背离（价格新低但RSI不新低）= 关注反转

## 五、七轨布林线

### 计算公式

```python
import pandas as pd
import numpy as np

def calc_7track_boll(df, n=20, k=5):
    """
    七轨布林线计算
    参数:
        df: DataFrame, 需含 'close' 列
        n: 均线周期，默认20
        k: 标准差平滑周期，默认5
    返回:
        df 新增7列: boll, top, track1, track2, track4, track5, bottom
    """
    mid = df['close'].rolling(n).mean()
    std0 = df['close'].rolling(n).std()
    dev = std0.rolling(k).mean()  # 标准差的移动平均，平滑波动

    df['boll'] = mid                    # 中轨：20日均线
    df['top'] = mid + 3 * dev           # 顶轨：+3σ
    df['track1'] = mid + 2 * dev        # 一轨：+2σ
    df['track2'] = mid + 1 * dev        # 二轨：+1σ
    df['track4'] = mid - 1 * dev        # 四轨：-1σ
    df['track5'] = mid - 2 * dev        # 五轨：-2σ
    df['bottom'] = mid - 3 * dev        # 底轨：-3σ

    return df
```

### 七轨分区

| 轨道 | 位置 | 价格状态 | 操作含义 |
|------|------|---------|---------|
| 顶轨 (MID+3σ) | 极度超买区 | 历史性高位 | 🔴 极度危险，坚决不追 |
| 一轨 (MID+2σ) | 超买区 | 远离均线上方 | 🟡 偏贵，考虑减仓 |
| 二轨 (MID+1σ) | 偏强区 | 均线上方温和 | ⚪ 偏强但未过热 |
| BOLL (MID) | 中轨 | 价值中枢 | ⚪ 均值回归参考线 |
| 四轨 (MID-1σ) | 偏弱区 | 均线下方温和 | ⚪ 偏弱但未超卖 |
| 五轨 (MID-2σ) | 超卖区 | 远离均线下方 | 🟢 关注止跌信号 |
| 底轨 (MID-3σ) | 极度超卖区 | 历史性低位 | 🟢🟢 如果基本面OK，可能是黄金坑 |

### 七轨布林线 vs 标准布林带

**核心区别**：
1. **标准差平滑**：DEV = MA(STD(C,20), 5) 而非直接用 STD(C,20)，轨道更平滑、假突破更少
2. **七轨细分**：将 -3σ 到 +3σ 的空间用 7 条线分成 6 个区域，比标准 3 轨（上中下）定位更精确
3. **内置K线**：原版同花顺代码将K线叠加在布林轨道内，视觉上直接看到价格在轨道中的位置

### 价值投资者的七轨布林用法

**核心用法**：
- 基本面确认好公司后，等价格触及四轨/五轨时建仓（不追高）
- 加仓信号：价格从五轨/底轨反弹 + 基本面未恶化 = 加仓机会
- 减仓信号：价格触及一轨/顶轨 + 估值偏高 = 考虑减仓
- 背离信号：价格创新低但布林带宽度收窄（波动率降低）= 变盘在即

## 六、成交额与换手率

### 成交额分析

**放量**：
- 成交额大幅增加
- 含义：市场关注度高
- 操作：关注方向

**缩量**：
- 成交额大幅减少
- 含义：市场关注度低
- 操作：等待变化

**天量天价**：
- 成交额创新高 + 价格创新高
- 含义：可能阶段性顶部
- 操作：谨慎

**地量地价**：
- 成交额创新低 + 价格创新低
- 含义：可能阶段性底部
- 操作：关注机会

### 换手率分析

| 换手率 | 含义 | 操作 |
|--------|------|------|
| <1% | 冷门股 | 流动性差，不适合大资金 |
| 1-3% | 正常 | 正常操作 |
| 3-5% | 活跃 | 关注方向 |
| >5% | 异常活跃 | 谨慎，可能见顶/底 |

### 量价配合

**量价齐升**：
- 价格上涨 + 成交额增加
- 含义：健康上涨
- 操作：持有或买入

**量价背离**：
- 价格上涨 + 成交额减少
- 含义：上涨乏力
- 操作：谨慎

**量价齐跌**：
- 价格下跌 + 成交额减少
- 含义：下跌动能减弱
- 操作：关注止跌信号

### 价值投资者的量价用法

**核心用法**：
- 不追放量大涨（可能见顶），偏好缩量回调时建仓
- 长期缩量横盘 + 基本面优秀 = 等待催化剂
- 天量天价 → 可能是阶段性顶部

## 七、融资融券数据

### 融资融券指标

**融资余额**：
- 投资者借钱买入的金额
- 增加：杠杆资金看多
- 减少：杠杆资金看空

**融券余额**：
- 投资者借券卖出的金额
- 增加：做空力量增强
- 减少：做空力量减弱

**融资/融券比**：
- 融资余额 / 融券余额
- >10：多头主导
- <3：空头主导

**融资买入占比**：
- 融资买入额 / 当日成交额
- >15%：杠杆资金活跃，波动加大

**融资担保比例**：
- 维持担保比例
- <150%：风险预警

### 融资融券分析

**融资余额趋势**：
- 持续增加：杠杆资金看多
- 持续减少：杠杆资金看空
- 突然变化：关注原因

**融券余额趋势**：
- 持续增加：做空力量增强
- 持续减少：做空力量减弱
- 突然变化：关注原因

**融资/融券比**：
- 高比值：多头主导，可能见顶
- 低比值：空头主导，可能见底

### 价值投资者的两融用法

**核心用法**：
- 融资余额过高 → 杠杆资金聚集，一旦回调跌幅可能放大（风险信号）
- 融券余额异常增加 → 有人在做空，需关注基本面是否真的恶化
- 融资余额极低 → 杠杆出清，可能接近底部（配合基本面判断）

### 信号冲突处理规则

| 冲突情况 | 处理方式 |
|---------|---------|
| 超买+超卖信号同时出现 | 以趋势信号为主（均线>MACD>RSI>布林） |
| 短期超买+长期超卖 | 分批建仓，短期等回调，长期看多 |
| 技术面看空+基本面看多 | 以基本面为主，技术面仅调整建仓节奏 |
| 所有信号一致 | 高置信度，可加大仓位 |
