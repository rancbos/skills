# 银行股分析要点与 Pitfall

> 银行股分析与一般制造业/消费股有显著差异。本文档记录银行股分析的特殊要点。

## 一、银行股核心指标体系

### 盈利能力
| 指标 | 含义 | 优劣判断 | 数据来源 |
|------|------|---------|---------|
| 净息差 NIM | 利息净收入/生息资产平均余额 | >2%优秀，1.6-2%良好，<1.5%承压 | 年报"管理层讨论"章节 |
| 净利差 | 生息资产收益率-计息负债成本率 | 与NIM趋势一致 | 年报 |
| 成本收入比 | 业务管理费/营业收入 | <30%优秀，30-35%良好，>35%偏高 | 年报主要财务指标表 |
| ROE(加权) | 归母净利/加权平均净资产 | >12%优秀，10-12%良好，<10%一般 | 年报/季报 |
| ROA | 净利润/平均总资产 | >0.8%优秀，0.6-0.8%良好 | 年报 |

### 资产质量（银行分析核心）
| 指标 | 含义 | 优劣判断 | 注意事项 |
|------|------|---------|---------|
| 不良贷款率 | 不良贷款/贷款总额 | <1%优秀，1-1.5%良好，>1.5%关注 | 关注趋势比绝对值更重要 |
| 拨备覆盖率 | 贷款减值准备/不良贷款 | >250%优秀，200-250%良好，<150%危险 | 越高安全垫越厚，但可能隐藏利润 |
| 拨贷比 | 贷款减值准备/贷款总额 | >2.5%良好 | 与拨备覆盖率配合看 |
| 关注类贷款占比 | 关注类贷款/贷款总额 | <2%良好 | 预警指标，关注迁移率 |
| 对公/零售不良率 | 分类不良率 | 需分别分析 | 零售不良率上升是常见风险信号 |

### 资本充足率
| 指标 | 监管要求 | 安全边际 | 注意事项 |
|------|---------|---------|---------|
| 核心一级资本充足率 | ≥7.5%（含储备资本后≥8.5%） | >9%安全 | 高速扩张时消耗快 |
| 一级资本充足率 | ≥8.5% | >10%安全 | — |
| 资本充足率 | ≥10.5% | >12%安全 | 包含二级资本债等 |

## 二、银行股估值方法

### PB估值法（首选）
- 银行股最常用估值方法
- PB = 股价 / 每股净资产
- 合理PB与ROE正相关：ROE 12% → PB 0.7-1.0x；ROE 15% → PB 1.0-1.3x
- 破净（PB<1）不一定便宜，需看ROE能否维持

### PE估值法（辅助）
- PE = 股价 / EPS
- 城商行PE通常4-8倍
- 低PE可能反映市场对增长可持续性的怀疑

### 股息率估值
- 银行股普遍高分红，股息率是重要参考
- 股息率 > 4% 具有吸引力（在低利率环境下）

## 三、银行股分析特殊 Pitfall

- **毛利率不适用**：银行没有传统意义的毛利率，用NIM替代
- **现金流分析受限**：银行经营现金流波动极大（受存贷款变动影响），不能简单用"经营CF/净利>1"判断利润质量
- **资产负债率高是正常的**：银行资产负债率90%+是行业特性，不能用一般企业标准判断
- **净利润增速 vs 营收增速**：银行净利润增速可能远高于营收增速（因为成本压降、拨备释放等），需分别分析驱动因素
- **分板块分析**：对公/零售/资金业务的不良率、收益率、增速差异大，必须分开看
- **永续债影响**：部分银行发行了永续债（其他权益工具），计算EPS和ROE时需扣除永续债利息
- **审计意见**：银行年报同时有境内（中国会计准则）和境外（IFRS）两套审计，均应为无保留意见

## 四、数据提取要点

### 年报关键章节
1. **第二节 主要会计数据和财务指标** → 营收/净利/ROE/NIM/不良率/拨备覆盖率/资本充足率
2. **第五节 管理层讨论与分析** → 资产质量详情、分业务数据、贷款结构
3. **第八节 股份变动及股东情况** → 股东结构、质押情况
4. **附注** → 详细财务报表

### pdftotext 提取命令
```bash
# 提取主要财务指标
pdftotext "年报.pdf" - | grep -A 2 -i "营业收入\|净利润\|不良贷款率\|拨备覆盖率\|资本充足率\|净息差\|每股净资产"

# 提取股东信息
pdftotext "年报.pdf" - | grep -B 1 -A 2 "海尔\|国信\|意大利\|圣保罗\|总股本\|控股股东"

# 提取管理层
pdftotext "年报.pdf" - | grep -A 3 "董事长\|行长\|副行长"
```
