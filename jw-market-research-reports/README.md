# jw-market-research-reports

市场研究报告

## 简介

生成全面的市场研究报告，支持LaTeX→PDF和飞书云文档两种输出格式。风格对标顶级咨询公司（麦肯锡、波士顿咨询、高德纳）。具备可视化生成（scientific-schematics/generate-image）、多源数据收集（research-lookup/内置搜索/金十数据）、多框架战略分析（波特五力、PESTLE、SWOT、TAM/SAM/SOM、波士顿矩阵）。
