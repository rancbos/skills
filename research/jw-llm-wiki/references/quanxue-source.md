## URL 模式

每本书的索引页：
```
https://www.quanxue.cn/ct_<category>/<bookname>index.html
```

章节页：
```
https://www.quanxue.cn/ct_<category>/<bookname>/<bookname><NN>.html
```

示例（管子）：
- 索引: `https://www.quanxue.cn/ct_fajia/guanziindex.html`
- 第01章: `https://www.quanxue.cn/ct_fajia/guanzi/guanzi01.html`
